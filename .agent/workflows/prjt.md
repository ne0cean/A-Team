---
description: 프로젝트별 작업 진행 현황 및 목록 요약 (Central TODO 기반)
---

1. `TODO.md` 파일을 읽어 전체 작업 목록을 스캔 (A-Team 루트 기준).
2. **[최우선]** `## 📋 Task List` 섹션에서 완료되지 않은 작업(`[ ]`)들을 추출하여 브리핑의 맨 상단에 리스트 형태로 먼저 나열.
3. 그 다음, 각 프로젝트별로 대기 중인 작업 개수와 최근 완료된 작업을 취합하여 표(Table) 형태로 브리핑.
4. 사용자에게 현재 추적 중인 프로젝트의 전반적인 상태를 요약하여 보고.
5. (선택) 특정 프로젝트의 상세 내용을 보고 싶으면 `/todo [프로젝트명]`을 입력하도록 안내.
