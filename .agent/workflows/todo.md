---
description: 추가 및 조회를 위한 프로젝트 TODO 관리 (GitHub 동기화 포함)
---

1. `TODO.md` 파일을 읽어 현재 상태 확인 (A-Team 루트 기준).
2. 사용자가 `/todo [내용]`을 입력한 경우:
   - 현재 작업 중인 디렉토리명(예: `connectome`)을 대괄호 `[]`로 감싸 말머리에 붙여 `TODO.md`의 `## 📋 Task List` 아래에 추가.
   - `git commit` 및 `git push`를 통해 깃헙 동기화 시도.
   - "✅ 중앙 TODO에 추가되었습니다." 메시지와 함께 현재 목록 출력.
3. 사용자가 `/todo`만 입력한 경우:
   - `TODO.md`에서 아직 완료되지 않은 (`[ ]`) 모든 작업 목록을 불러옴.
   - 사용자에게 브리핑.
4. (선택) 체크 이미지나 리스트 형태로 깔끔하게 브리핑.
