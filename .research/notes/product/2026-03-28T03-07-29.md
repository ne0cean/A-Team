**DB 선택지 비교**:

| 옵션 | 비용 | 복잡도 | 추천 |
|------|------|-------|------|
| Railway PostgreSQL | 무료 tier 500MB | 중 | ★★★ |
| PlanetScale (MySQL) | 무료 tier 5GB | 중 | ★★☆ |
| Upstash Redis | 무료 tier 10K cmd/day | 소 | ★★★ |
| lowdb (로컬 JSON) | 무료 | 소 | 개발용만 |

**Upstash Redis 추천 이유**: Room Settings는 단순 Key-Value이므로 HSET roomCode → {name, description, createdAt} 패턴이 완벽하게 맞음. Railway 재시작과 무관하게 영속.

**추가할 Socket.io 이벤트**:
```
client → server: update_room_settings { name, description }
server → client: room_settings_updated { name, description }
```

**예상 난이도**: 중 (4-6h)

**TURN 서버 옵션**:

| 옵션 | 비용 | 복잡도 |
|------|------|-------|
| Metered.ca | 0.4$/GB | 낮음 (클라우드) |
| Twilio Network Traversal | 0.4$/GB | 낮음 (클라우드) |
| coturn (셀프호스팅) | VPS 비용만 | 높음 |
| Xirsys | 무료 500MB/월 | 낮음 |

**추천**: Xirsys 무료 플랜 시작 → 트래픽 증가 시 Metered.ca 전환

**iceServers 설정 예시**:
```js
const iceServers = [
  { urls: 'stun:stun.l.google.com:19302' },
  {
    urls: 'turn:turn.xirsys.com',
    username: process.env.TURN_USER,
    credential: process.env.TURN_PASS,
  }
];
```

**예상 난이도**: 중 (3-5h) — 계정 설정 + env 연결 + 테스트

### 아이디어 2: Ambient Status Wall (주변 상태 흐름)

**컨셉**: 공간 내 사람들의 shout/reaction을 시간순 피드로 보여주는 사이드 패널

**핵심 기능**:
- 우측 슬라이드인 패널 (너비 240px)
- 최근 20개 shout/reaction 스트림 (페이드인/페이드아웃)
- 각 항목: 아바타 색 + 닉네임 + 내용 + 상대 시간 ("3초 전")
- 프라이버시 Tier 1 (익명)은 색상 점만 표시

**차별화**: 공간이 **살아있다는 감각** — 아무도 말 안 해도 누군가 리액션하고 있다는 ambient awareness

**구현 난이도**: 소~중 (3-5h)
- `useStore.js` — `activityFeed: Array<{type, userId, content, ts}>` 상태 추가
- SpacePage/RoomPage — socket 이벤트 수신 시 feed 업데이트
- `ActivityFeed.jsx` 신규 컴포넌트 (AnimatePresence 리스트)

**기존 아키텍처 영향**: 낮음 — 기존 이벤트 재활용

### 아이디어 3: Encounter Card (만남 기록)

**컨셉**: 같은 공간에서 일정 시간 이상 함께한 사람에게 "오늘 스쳤어요" 카드 생성

**핵심 기능**:
- 근접 30초 이상 유지 시 "스침 감지" (로컬 타이머)
- 세션 종료 시: "오늘 N명과 스쳤어요" 요약 모달
- 각 카드: 아바타 색 + 닉네임(공개 시) + 함께한 시간 + 공간명
- 옵션: 상대방에게 "연결 요청" → DM으로 이어짐

**차별화**: 오프라인 세계의 **우연한 만남을 디지털화** — "지나쳤던 사람" 감성을 lightweight spatial presence로 구현

**구현 난이도**: 중~대 (6-10h)
- 근접 타이머: `useCanvasInteraction.js`의 근접 감지 로직 확장
- `encounterMap: Map<userId, { duration, nickname, avatar }>` 상태
- 세션 종료 감지: `beforeunload` + Socket.io `disconnect`
- `EncounterSummary.jsx` 모달 컴포넌트
