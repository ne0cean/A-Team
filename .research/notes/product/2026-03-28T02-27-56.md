## 현재 기능 인벤토리

### 진입 방식 (App.jsx)
- 3가지 시작 경로: GPS 모드, 데모 모드, 프라이빗 룸 모드
- 프로필 설정: 닉네임, 상태메시지, 아바타 색상(6가지), localStorage 저장
- 데모 모드: 초대 없이 DEMO00 공간에 빠른 진입 (Guest_XXXX)

### GPS 기반 모드 (SpacePage)
- 지리 위치 기반: 실시간 GPS 수집 (throttle 2s), 반경 내 사용자 렌더링
- 캔버스 상호작용: 드래그 이동, 줌 인/아웃, 더블탭 ping, 싱글탭 프로필 수정
- 데모 지원: 고정 좌표(서울 37.5665, 126.9780)

### 프라이빗 룸 모드 (RoomPage)
- 폐쇄 공간: 초대 코드 기반 접근
- 공유: QR코드 + 초대 코드 복사 (1-tap)
- 실시간 카운트: 참여 인원 표시

### 공간 내 상호작용

#### 시각화
- Proximity Aura: 거리에 따른 스케일링 (1.0 → 1.15)
- Connection Line: 가장 가까운 사용자에게 연결선
- Cluster Fan-Out: 겹치는 아바타 자동 부채꼴 배치
- Ping Wave: 더블탭 시 원형 파동 애니메이션

#### 커뮤니케이션
- Reaction: 이모지 폭발 애니메이션
- Ephemeral Shout: 시간 제한 말풍선 (max 60자)
- DM: 1:1 메시지 (요청 → 수락 → 채팅)

#### 개인화
- 프라이버시 4단계: 익명(색상) → 닉네임 → 상태 → 연결 허용
- 기분: 6가지 이모지
- 프로필: 닉네임, 상태, 아바타 실시간 동기화

### 음성 채팅
- WebRTC 근접 통화: 자동 피어 매칭 (ID 사전순)
- 공간 패닝: Web Audio API (거리 기반 좌/우 스테레오)
- Mic 토글: 음소거 상태 시각화

### 알림 & 피드백
- Push: DM 요청, 사용자 입장
- Audio FX: 조인, 핑, 말풍선, DM 사운드
- 연결 상태: connecting → connected → reconnecting → offline

### PWA
- Manifest, Icons (192x192, 512x512), Standalone 모드, Auto-update

## 신규 기능 아이디어 (3가지)

### 1. Presence Pressure Gauge (근접 압력계)
**난이도**: 소 (2-3h)

**핵심 기능**: 주변 사용자 밀도를 실시간 원형 게이지로 표시
- 가까운 사람 많음 = 따뜻한 색 (빨강)
- 적음 = 차가운 색 (파랑)
- 혼잡도 정량화 → "이 공간이 얼마나 붐비는가" 즉시 감각

**차별화**: lightweight spatial presence의 핵심 = 밀집도 감각을 수치화 없이 색으로 전달

**구현 방향**:
- useCanvasInteraction 거리 분포 활용 (재사용)
- PressureGauge.jsx (SVG 원형)
- HSL color interpolation (cool→warm)
- requestAnimationFrame 기반 업데이트

---

### 2. Temporal Breadcrumb Trail (시간 흔적)
**난이도**: 중 (4-6h)

**핵심 기능**: 지난 2분 움직임을 흐릿한 궤적으로 표시
- 나의 경로 (파랑, 불투명 100%→0%)
- 다른 사용자와 경로 교차점 (자주색 하이라이트)
- Privacy-first: 내 경로는 로컬만 저장

**차별화**: 정적 위치가 아닌 "동선 흐름" → 무언의 만남 감각 생성

**구현 방향**:
- 위치 히스토리 링 버퍼 (120개 = 2분×60fps)
- Canvas drawImage 기반 페이딩 처리
- 서버 전송 없이 로컬 계산만

---

### 3. Mood Contagion Heatmap (감정 전파 맵)
**난이도**: 대 (8-12h)

**핵심 기능**: 사용자 기분을 공간 히트맵으로 렌더
- 긍정 (🔥💫🎉) = 빨강/주황
- 부정 (😢😤) = 파랑/보라
- 밀도 높음 = 더 강한 열감
- 실시간 서버 전파

**차별화**: 감정의 ambient display → 언어 없는 집단 감정 공유

**구현 방향**:
- 서버: mood 집계 + grid cell 전파
- Canvas gradient + requestAnimationFrame
- 프라이버시 티어 연동 (tier < 2 = 익명 기여)
