## 신규 기능 아이디어

### 아이디어 1: Vibe Zones (기분 기반 그룹화)
- 핵심 기능: 같은 mood 사용자들을 클러스터링하여 아우라 색상으로 강조. 기분별 사용자 카운터 표시.
- 차별화: 집단 감정 다이나믹스 시각화. Serendipity 강화. 기존 Mood 기능의 자연스러운 확장.
- 난이도: **소-중** (1-2h). Zustand selector 추가 + Avatar 아우라 동적화만으로 구현 가능. 새 Socket 이벤트 불필요.

### 아이디어 2: Breadcrumb Trail (이동 경로 자국)
- 핵심 기능: 지난 30초 사용자 이동 경로를 페이드아웃 선으로 캔버스에 표시.
- 차별화: Spatial presence를 가장 직관적으로 시각화. 모바일 입력 제약 극복. 존재감 강화.
- 난이도: **중** (3-4h). Canvas drawing API + 히스토리 배열 + requestAnimationFrame. 모바일 성능 주의.

### 아이디어 3: Nearby Suggestions (지능형 매칭)
- 핵심 기능: "근처 5명 중 2명 추천" + 호환성 점수(거리+티어+기분). 1클릭 DM/Ping.
- 차별화: 반자동 연결로 수동 상호작용 부담 감소. Privacy 티어 존중한 매칭.
- 난이도: **중-대** (4-6h). 호환성 알고리즘 설계 + 추천 UI 패널 필요.
