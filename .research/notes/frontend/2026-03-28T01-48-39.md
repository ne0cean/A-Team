- **파일**: `src/pages/RoomPage.jsx` line 79-82

- **난이도**: 보통
```js
const usersWithOffsets = useMemo(() =>
  users.map((u) => {
    const offset = clusterOffsets[u.id];
    return offset ? { ...u, ...offset } : u; // offset 없는 경우 u 그대로 반환 (참조 유지 안됨)
  }), [users, clusterOffsets]
);
// → 개선: users를 Map으로 관리하거나 offset 변경 user만 새 객체 생성
```

### 6. **[중간]** Zustand updateUserPosition 선택적 업데이트

- **변경**: 좌표가 실제로 변경된 경우만 새 객체 생성
- **효과**: 동일 위치 emit 시 불필요한 re-render 방지
- **난이도**: 쉬움
```js
updateUserPosition: (id, x, y) => set((state) => {
  const user = state.users.find(u => u.id === id);
  if (!user || (user.x === x && user.y === y)) return state; // no-op
  return { users: state.users.map(u => u.id === id ? { ...u, x, y } : u) };
}),
```
