### 핵심 차별점 3가지
1. **마찰 제로 입장**: 계정 없음, 비밀번호 없음 → 링크/QR 스캔 즉시 입장
2. **프라이버시 주도권**: 내가 얼마나 드러낼지 4단계로 실시간 조절
3. **위치 기반 우연성**: GPS로 지금 이 공간의 낯선 사람을 만나는 경험 (오프라인 감각)

## 단계 2 — 포지셔닝 분석

### 2026년 "virtual presence" 시장 지형
| 플레이어 | 포지션 | Vibe Here 비교 |
|---|---|---|
| Gather.town | 팀/이벤트 2D 공간, 계정 필요, 유료 | 더 무겁고 업무 중심 |
| Discord Stage / VC | 오디오 중심, 커뮤니티 기반, 계정 필수 | 텍스트/음성 위주, 공간감 없음 |

| **Vibe Here** | **링크 하나로 2D 공간, 계정 없음, 경량** | **틈새: 즉석/일회성/장소 기반** |

### 공백 포지션
> **"가벼운 위치 기반 실시간 공존"** — 앱 설치도, 계정도, 설정도 없이 지금 이 공간에서 다른 사람의 존재를 느끼는 것

### 타겟 분석
| 세그먼트 | 핵심 니즈 | 진입 장벽 |
|---|---|---|
| **개발자팀 / 원격 협업** | "지금 누가 일하고 있지?" 존재감 공유 | 낮음 (링크 공유) |
| **오프라인 이벤트 / 밋업** | 행사장에서 같은 공간 사람 연결 | 낮음 (QR 스캔) |
| **원격 스터디 그룹** | 같이 공부하는 느낌, 가벼운 접속 | 낮음 |
| **소규모 커뮤니티 / 클래스** | 강사↔학생 비언어 소통 공간 | 낮음 |
| **카페/공유공간 커뮤니티** | GPS로 "지금 카페에 있는 사람" 만나기 | GPS 허용 필요 |

**최우선 타겟**: 개발자팀 + 원격 스터디 (B2B 아닌 바텀업, 바이럴 가능성 최고)

### 대안 태그라인 3개
1. **"링크 하나로 같은 공간에"** ← 현재 방향, 프라이빗 룸에 최적
2. **"계정 없이, 지금 당신 주변의 사람들"** ← GPS 모드 강조, 오프라인 감각
3. **"가장 가벼운 버추얼 오피스"** ← 원격팀 타겟, B2B 어필

**추천**: 태그라인을 모드별로 분기
- GPS 모드 랜딩: *"지금 이 공간, 30m 안의 사람들을 만나세요"*
- 프라이빗 룸: *"링크 하나로 같은 공간에"*
- 일반: *"계정 없이 시작하는 실시간 공간"*

## 단계 4 — 배포 채널 전략

### 얼리어답터 도달 채널 5가지

#### 1. Product Hunt (론칭)
- **타겟**: 글로벌 얼리어답터, 빌더, 인디해커
- **접근**: 프라이빗 룸 기능을 "팀 프레젠스 레이어"로 포지셔닝
- **메시지 톤**: 기술적 + 놀라움 → "No login, no friction. Just presence."
- **전술**: 론칭 당일 Demo 룸 링크를 Product Hunt 코멘트에 공유, 실시간 방문자 수 보여주기
- **타이밍**: 화요일~목요일 PST 기준 오전

#### 2. Hacker News (Show HN)
- **타겟**: 개발자, 테크 얼리어답터
- **접근**: `Show HN: Vibe Here – Real-time 2D space with no signup required`
- **메시지 톤**: 기술 구현 중심, 솔직함 → WebSocket + GPS + 4-tier privacy 구조 설명
- **전술**: 직접 Demo 룸 링크 첨부, 개발 동기(friction 제거) 솔직히 설명
- **주의**: 마케팅 언어 금지, "I built this" 톤 유지

#### 3. 개발자 커뮤니티 (한국)
- **채널**: 개발자 카카오톡 오픈채팅, 주니어 개발자 Discord, 우아한형제들/토스 테크 블로그 독자층
- **타겟**: 국내 개발자, 원격 스터디 그룹
- **접근**: "원격 스터디할 때 같이 접속하는 공간" 시나리오로 접근
- **메시지 톤**: 친근한 한국어, 실용적 → "스터디 룸 링크 하나로 공유하면 다 같이 캔버스에 나타나요"
- **전술**: 공개 스터디 룸 링크를 아예 공개 채널에 올려서 바이럴 유도

#### 4. Twitter/X + LinkedIn (빌더 네트워크)
- **타겟**: 인디해커, 원격팀, 스타트업 팀장
- **접근**: 제품 데모 GIF/영상 (말풍선 + 아바타 이동 + QR 공유 장면)
- **메시지 톤**:
  - Twitter: 짧고 임팩트 → "What if Zoom had no friction? Demo:" + 링크
  - LinkedIn: 원격 협업 문제→솔루션 프레이밍
- **전술**: 동시 접속자가 많을 때 스크린캐스트 트윗 ("지금 XX명이 같은 공간에")

#### 5. 오프라인 이벤트 / 밋업 연계
- **타겟**: 개발자 밋업, 해커톤 참가자
- **접근**: 행사 당일 GPS 모드 활성화 → "이 행사에 온 사람들을 여기서 만나세요" QR 스티커 제공
- **메시지 톤**: 물리적 공간 + 디지털 공간 연결 경험
- **전술**: 행사장 입구에 QR 코드 스티커 부착, 주최자에게 "GPS 기반 밋업 공간" 무료 제공

## 핵심 발견 요약

1. **차별점 1위**: "계정 없는 즉시 입장"이 가장 강력한 훅 — 경쟁 제품 모두 계정 필요
2. **GPS 모드는 독보적**: 물리 공간↔디지털 공간 연결은 현재 시장에 직접 경쟁자 없음

5. **Product Hunt 론칭이 최우선**: 데모 룸 실시간 동접자 수 활용 가능 (사회적 증거)
