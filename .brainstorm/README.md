# .brainstorm/ — 브레인스토밍 라운드 저장소

`/brainstorm` 호출 시 자동 생성되는 라운드별 결과 저장 디렉토리.

## 구조

```
.brainstorm/
  rounds/
    round-1-2026-05-21.md
    round-2-2026-05-21.md
    ...
  README.md
```

## 파일 형식

각 라운드 파일은 3섹션 포함:
- EXPAND 결과 (4 퍼소나)
- ATTACK 결과 (Mom Test / 9차원 / Reality Score)
- SYNTHESIZE 판정 (Judge 통합)
