# k-skill — 한국 특화 AI 스킬 모음집

> 70+ 한국 서비스 자동화 스킬. 필요한 기능이 있을 때 이 레포에서 찾아서 가져다 쓸 수 있음.

## 레포 정보
- **GitHub**: https://github.com/NomaDamas/k-skill
- **타겟**: 한국 사용자 특화 서비스
- **스킬 수**: 70+
- **지원 에이전트**: Claude Code, Codex, OpenCode, OpenClaw

## 주요 카테고리

### 1. 교통/여행
- SRT/KTX 기차 예약 및 관리
- 산림휴양림 숙박시설 공실 검색
- 배송 추적 (주요 택배사)

### 2. 금융/법률
- 한국 주식시장 정보
- 특허/상표 검색
- 증권 계좌 모니터링
- 부동산 실거래가 조회
- 법원 경매 부동산 검색
- 법인등기 상담

### 3. 쇼핑/이커머스
- 쿠팡 상품 검색 (필터링)
- 중고차 시세 비교
- 재고 확인 (다이소, 올리브영, 마켓컬리)

### 4. 정보/유틸리티
- 날씨 예보 + 미세먼지
- 지하철 도착 정보
- 로또 당첨 확인
- 한국사 기록 검색

### 5. 문서 처리
- HWP 문서 변환/편집 (한글 네이티브 포맷)
- 맞춤법/문법 검사
- 한국어 특화 문자 카운팅

## 기술 스택
- Node.js + Python 패키지
- HTTP 기반 API 프록시 (`k-skill-proxy`)
- 추가 클라이언트 API 레이어 불필요

## 인증 요구사항
| 레벨 | 설명 | 예시 |
|------|------|------|
| Required | 사용자 직접 인증 필요 | SRT/KTX 예약, 증권 계좌 |
| Optional | 운영자 키로 기능 향상 | - |
| Not required | 공개 API 기반 | 날씨, 미세먼지, 로또 |

## 사용 시나리오

**언제 참고하나**:
- 한국 특화 서비스 자동화 요청 시
- 교통/금융/법률 데이터 필요 시
- HWP 문서 처리 필요 시
- 한국어 NLP 기능 필요 시

**어떻게 쓰나**:
1. 필요한 기능 확인 (위 카테고리에서 검색)
2. k-skill 레포에서 해당 스킬 찾기
3. A-Team 프로젝트에 선택적으로 통합 (전체 X)

## 통합 방침
- ✅ **참고용**: 필요 시 개별 스킬 가져오기
- ❌ **전체 통합 금지**: 70+ 스킬 전체 설치 안 함
- ✅ **선택적 차용**: 특정 기능만 `.claude/commands/` 또는 스크립트로 이식
