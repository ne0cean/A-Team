# Preset: Local Event — 카페/바/밋업 이벤트 페이지

> 오프라인 이벤트 고지. 주최자가 직접 만든 느낌. SNS 공유 목적.

## 대표 레퍼런스 패턴

| 출처 유형 | 예시 | 왜 효과적인가 |
|----------|------|-------------|
| 인스타 링크 in bio | 카페 이벤트 대부분 | "인스타에서 왔구나" 맥락 |
| Notion 페이지 | 소규모 밋업 | 표+이미지, 편집 쉬움 |
| 카카오 채널 게시글 | 한국 로컬 | 익숙한 포맷 |
| Eventbrite 무료 | 해외 밋업 | 신뢰 + 참가 신청 |

## 해부 결과 (공통 패턴)

```yaml
font:
  heading: "시스템 폰트 또는 Pretendard"
  body: "동일"
  sizes: [20px, 15px, 13px]
  특징: "제목이 생각보다 크지 않음 (20-24px)"

color:
  palette: ["#222", "#fff", 포인트 1개]
  accent: "이벤트 성격에 따라 다름 (카페=갈색/초록, 바=검정/금)"
  특징: "배경이 흰색 아닌 경우 많음 (크림, 연갈색)"

layout:
  구조: "1단 세로, 모바일 우선"
  전형적 순서:
    1: "이벤트 이름 (크게)"
    2: "날짜 + 시간 + 장소 (눈에 띄게)"
    3: "이벤트 설명 3-5줄"
    4: "사진 1-2장 (장소 실사)"
    5: "참가 방법 (링크 또는 DM)"
    6: "주최자 한 줄 소개"
  특징: |
    - 날짜/시간/장소가 상단 1/3 안에 반드시 있음
    - 지도 임베드 또는 주소 텍스트
    - 스크롤 짧음 (모바일 3-4 스크린)

copy_tone:
  스타일: "반말 또는 존댓말 (카페=존댓말, 바=반말)"
  특징: |
    - "함께해요" / "놀러오세요" 톤
    - 시간 포맷: "5/25 (토) 오후 3시"
    - 가격: "무료 / 음료 1잔 주문" 같은 실질 정보
    - 해시태그 1-3개 하단

imagery:
  유형: "실제 장소 사진 (폰으로 찍은 느낌)"
  특징: |
    - 인스타 감성 보정 가능하지만 과하지 않게
    - 메뉴/공간 사진 > 사람 사진
    - 포스터 이미지가 있으면 그게 메인
```

## Intentional Imperfections

```yaml
필수 반영:
  - 사진 비율 통일 안 됨 (세로+가로 혼재)
  - "문의: @인스타아이디 DM" 같은 연락처
  - 정확한 주소보다 "xx역 3번 출구 도보 5분"
  - 참가비가 "5,000원 (음료 포함)" 같은 현실적 가격
  - 하단에 "※ 주차 불가" 같은 실무 공지
  - 수정 이력 흔적 ("*장소가 변경되었습니다" 같은)
  - 날짜 지나면 "마감되었습니다" 수동 표시

선택 반영:
  - 카카오맵 링크 (네이버지도 아님)
  - 참가 인원 표시 "현재 12/20명"
  - 댓글/문의 섹션이 구글폼 링크
```

## Anti-Pattern

```yaml
절대 하지 말 것:
  - "Join our exclusive event" 같은 영어 카피
  - 타이머/카운트다운 위젯
  - 발표자 프로필 카드 (대기업 컨퍼런스 느낌)
  - 스폰서 로고 나열
  - 티켓 등급 (VIP/일반) 구분
  - 자동 리마인더 시스템
  - 세련된 포스터 디자인 (Canva 프리미엄 느낌)
```
