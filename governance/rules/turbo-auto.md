# Turbo & Auto-Proceed Rules

에이전트가 사용자 개입 없이 자율적으로 작업을 완수하기 위한 지침.

**적용 범위**: 모든 에이전트의 일반 자율 ���행. 사용자가 대화 중일 때도 적용.
**관계**: `autonomous-loop.md`는 수면/야간 ��드 전용 계약(체크포인트·PID 락·나레이션 금지 등). 이 문서와 독립적으로 적용.

## 1. SafeToAutoRun
시스템 탐색(`ls`, `cat`), 상태 확인(`git status`), 테스트 실행 등 위험도가 낮은 명령은 승인 없이 자동 실행.

## 2. 연속적 도구 호출
[분석 → 수정 → 검증]을 한 턴에 묶어 연속 호출. 독립 호출은 병렬로.

## 3. 자율적 수정
실패 시 원인 파악 후 최소 2회 재시도. 2회 실패 시 사용자 에스컬레이션.
