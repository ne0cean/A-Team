# PPT 반-AI 라이팅 룰

업무용 PPT에서 AI 제작 느낌을 제거하는 핵심 원칙.
에이전트가 슬라이드 콘텐츠를 생성할 때 반드시 적용.

---

## 1. 헤드라인: 결론을 먼저

| AI 느낌 (금지) | 자연스러운 표현 |
|----------------|----------------|
| 매출 현황 | Q1 매출 23억원 — 목표 대비 +7% |
| 시장 동향 분석 | 경쟁사 3사 가격 인상, 당사 점유율 확대 기회 |
| 주요 성과 | 신규 고객 147개사 — 엔터프라이즈 비중 41%로 첫 과반 |
| 향후 방향성 | Q2 집중 영역: 해지율 저감 + 업셀 파이프라인 |
| 개선 필요 사항 | 배송 지연 3.2일 → 1.5일 목표, 8월까지 |

**규칙**: 헤드라인 = 그 슬라이드의 핵심 주장. 독자가 헤드라인만 읽어도 내용 파악 가능.

---

## 2. 불릿: 짧고 구체적으로

**최대 3개 불릿 / 슬라이드**
**불릿당 최대 30자** (한글 기준)

| AI 느낌 (금지) | 자연스러운 표현 |
|----------------|----------------|
| 다양한 채널을 통해 고객과의 접점을 강화하고 있습니다 | 카카오 채널 MAU 34만명 (+18% QoQ) |
| 전반적인 성과 지표가 개선되는 추세를 보이고 있습니다 | NPS 42점 → 61점, 업계 평균 상회 |
| 지속적인 노력을 통해 품질 향상을 추구하고 있습니다 | 불량률 2.1% → 0.8%, 목표치 달성 |

**금지 표현**:
- "~하고 있습니다" (현재진행형 남발)
- "다양한", "여러", "많은" (수량 미명시)
- "상당한", "획기적인", "혁신적인" (형용사 과장)
- "지속적으로", "전략적으로" (부사 공허)
- "추진 중", "검토 중" (진행 상태 모호)

---

## 3. 데이터: 반드시 구체적 수치

사용자가 데이터를 제공하지 않은 경우 `[DATA]` 플레이스홀더 사용. 임의 숫자 생성 금지.

```
[DATA: Q1 매출액 입력 필요]
[DATA: 목표 달성률 입력 필요]
```

단위 명시 필수: `23억원` (O) / `23` (X)
기간 명시 필수: `전기 대비 +23%` (O) / `+23%` (X)
비교 기준 명시: `업계 평균(78%) 대비` (O) / `업계 평균 대비` (X)

---

## 4. 레이아웃 다양성

슬라이드 3장 이상 동일 레이아웃 금지.

**권장 시퀀스 예시**:
```
cover → agenda → section_break → single → two_column → data_table → quote → closing
```

콘텐츠 유형별 레이아웃 선택 기준:
- 결론 1개 강조 → `single` (불릿 1-3개)
- 비교/대조 → `two_column`
- 숫자/실적 → `data_table`
- 핵심 메시지 강조 → `quote`
- 챕터 구분 → `section_break`

---

## 5. 한국어 비즈니스 문체

자연스러운 한국 기업 문서 어투 사용. 영어 직역 금지.

| 영어 직역 (금지) | 한국어 자연 표현 |
|-----------------|----------------|
| 핵심 퍼포먼스 인디케이터 | 핵심 성과 지표 (KPI) |
| 스테이크홀더 인게이지먼트 | 이해관계자 소통 |
| 리소스 얼로케이션 최적화 | 자원 배분 효율화 |
| 시너지 창출 | 협업 효과 |
| 밸류 프로포지션 | 차별화 가치 |

**제목 어미**: `~ 현황`, `~ 분석`, `~ 방안` — 명사형으로 마무리
**불릿 어미**: 명사형 또는 수치로 마무리 (동사형 지양)

---

## 6. 스피커 노트 활용

스피커 노트에는 슬라이드에 담지 못한 맥락과 논거를 작성:
- 데이터 출처 및 측정 기준
- 청중 예상 질문 및 답변
- 강조할 포인트 순서
- 전환 멘트

노트도 간결하게: 불릿 3-5개, 완전한 문장 불필요.
