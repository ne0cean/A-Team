# Arc Browser — Soft Pastel Experimental

**tone**: soft-pastel | **variant**: soft | **variance**: 6 | **motion**: 7 | **density**: 4

## Brand Essence

Browser as a creative space. 다채로운 profile 색상. 부드러운 유기적 곡선. "Browser for the internet we want".

## Typography

- Display: Söhne / custom display
- Body: Inter
- Scale: 13 / 14 / 16 / 18 / 22 / 32
- Line-height: 1.5

## Color Palette

- Primary: 유저가 선택한 profile color (gradient 가능)
- Background: 프로필별 gradient wallpaper
- Text: white on dark, dark on light (profile-aware)
- Accent: 채도 풍부한 파스텔 전범위

## Spacing

8px grid. Sidebar 중심 — 왼쪽 탭 스택.

## Motion

- Spring-based (macOS native feel)
- Tab 열기/닫기: smooth spring
- Spatial browsing: 공간 이동 metaphor
- Liveliness 강조 but not bouncy overshoot

## Components Signature

- Sidebar tab: rounded-lg (12px), hover/active 상태 차이 명확
- Command bar: 중앙 floating, 큰 padding
- Profile switch: gradient card 전환

## Anti-Patterns

- ❌ Sterile grayscale
- ❌ Dense data UI (Arc는 broad canvas)
- ❌ Corporate dark mode without character

## 언제 참고

- Creative tool, browser, 생산성 + 자아표현 결합
- 사용자별 커스터마이징이 주 기능일 때
- tone=soft-pastel + motion 7
