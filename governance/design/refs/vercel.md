# Vercel — Bold Typographic Industrial

**tone**: bold-typographic | **variant**: industrial | **variance**: 6 | **motion**: 5 | **density**: 5

## Brand Essence

Geist. Shipped. Performance-as-aesthetic. 흑백 + 대담한 타이포. 터미널 감성 + marketing polish.

## Typography

- Display / Body: **Geist** (Vercel open-source)
- Mono: **Geist Mono**
- Scale: 13 / 14 / 16 / 20 / 28 / 40 / 64 / 96+ (display 큼)
- Letter-spacing: tight on display (-0.03em)
- Line-height: 1.4 body, 1.0 display

## Color Palette

- Primary: `#000000` / `#FFFFFF` (pure)
- Surface: `#FAFAFA` / `#0A0A0A`
- Border: `#EAEAEA` / `#333333`
- Accent: 채도 높은 블루 `#0070F3` (단 한 번 사용)
- Geist color palette: slate scale 10 steps

## Spacing

8px grid. 섹션 간 대담한 여백. Hero는 100vh.

## Motion

- Transition: 150-250ms, `ease-in-out`
- Scroll-triggered reveal (marketing)
- Code deploy animation (signature interaction)
- Terminal typing effect (retro-futuristic touch)

## Components Signature

- Button: 32/40/48px, 6px radius, solid black/white, no shadow
- Card: 8px radius, 1px border, hover = border highlight
- Code block: Geist Mono, syntax highlight, copy button
- Geist UI components (오픈소스) — 참조 가능

## Anti-Patterns

- ❌ 보라 그라디언트 (Vercel 아이덴티티 반대)
- ❌ Soft pastel
- ❌ Bounce motion
- ❌ Emojis in marketing copy (dev tone 유지)

## 언제 참고

- DevTool, infra, deployment, performance-focused
- "ship fast + look sharp" 톤
- Geist 폰트 무료 사용 가능 (OFL)
