## Usage

```bash
# Taste evaluator reads this index for quick lookup
# Full DESIGN.md loaded on-demand per brand

# Find similar brands by primary color
grep '#ff' reference/design-systems/INDEX.md

# List all dark-canvas brands
grep '#0[0-9a-f]\{5\}' reference/design-systems/INDEX.md | grep Canvas
```

## Schema (per DESIGN.md)

```yaml
---
version: alpha
name: "{Brand}-design-analysis"
description: "1-paragraph brand essence"

colors:      # primary, canvas, ink, hairline, semantic-*
typography:  # display-xl..body-sm with fontFamily/size/weight/lineHeight/letterSpacing
spacing:     # base unit, section/card/component gaps
components:  # button, card, input, nav patterns
motion:      # duration, easing, transition patterns
---
```
