## 1. Bain — Strategic Planning That Produces Real Strategy (8장)

### 레이아웃 패턴

- **표지**: 사진 전면 배경 + 붉은 배너 오버레이 + 제목 white-on-red + 저자/부제 아래 배치. 세로 방향(portrait) 문서 포맷.
- **본문 페이지**: 2단 컬럼 텍스트 레이아웃이 기본. 섹션 헤드라인은 굵은 산세리프로 단독 행 차지.
- **피겨 슬라이드**: 텍스트 2단 + 하단 차트(막대그래프) 구조. 캡션 이탤릭체(`Figure 1:` 스타일). 차트 제목은 이탤릭 필기체(cursive) 폰트 사용.
- **마지막 슬라이드**: 연락처 정보를 테이블 형태(지역 | 담당자 이메일)로 정리 + 전 세계 오피스 목록을 소문자 foot line으로.
- **어바웃 페이지**: 붉은 스크립트 타이틀("Shared Ambition, True Results") + 본문 단락.

### 타이포그래피/컬러

- **주색**: Bain 레드 (#CC0000 계열), 링크/강조는 동일 레드
- **보조색**: 딥 네이비/다크 그레이 텍스트 (#1a1a1a)
- **배경**: 순백 (흰 종이 느낌, 여백 극대화)
- **헤드라인 폰트**: 굵은 산세리프 (Franklin Gothic/Arial Bold 계열)
- **본문 폰트**: 세리프체 (Georgia/Times 계열) — 12~13pt, 명확한 가독성
- **피겨 레이블**: 이탤릭 필기체 (Bain 시그니처 스타일)
- **강조 pullquote**: 붉은 배경 박스 + 흰 텍스트, 가운데 정렬 굵은 체

### 차트/다이어그램

- **막대 차트**: 단색(회색) 세로 막대. 데이터 레이블 막대 위/안에 직접 표기. X축 레이블 멀티라인. Source 표기 차트 하단 소문자 이탤릭.
- **범례**: 검은 사각형 범례 아이콘 + 텍스트 (차트 하단 좌측)
- **그룹화**: 막대 클러스터 사이 시각적 간격으로 그룹 구분
- **순환 다이어그램**: 화살표 연결 흐름도 (단색 파란 화살표, 회색 텍스트 박스 없이 플로팅 텍스트)
- **스캐터 플롯**: 점 레이블 직접 표기. 하이라이트 영역은 회색 박스.

## 2. Bain — Strategies for Growth (8장)

### 레이아웃 패턴

- **표지**: 세로 format. 중앙 상단 "BAIN & COMPANY" 로고 + 수직 구분선으로 2단 타이틀 분리 ("Strategies|for growth"). 좌하단 저자 이탤릭 박스. 우하단 하단 수평선 + 더 짧은 선으로 마감.
- **본문 2단**: 좌/우 동등 컬럼 분할. 헤드라인은 굵은 산세리프로 단독 라인, 컬럼 위에 floating.
- **피겨**: 본문 하단 또는 중간에 Figure 넘버링 + 차트 삽입. 피겨 캡션 이탤릭 굵은체 (`Figure N: 제목`).
- **강조 블록**: 컬럼 사이 또는 하단에 짙은 파란 좌측 보더라인 + 굵은 이탤릭 pullquote 텍스트.
- **마지막 페이지**: 회사 소개 단락 + Bain 로고 + 연락처 2단 레이아웃 + 도시 목록 footer.

### 타이포그래피/컬러

- **주색**: 딥 네이비 (#1d3a6b 계열) — 로고, 헤드라인, 강조
- **보조색**: 피겨 레이블에 같은 딥 네이비, 차트 막대는 네이비
- **배경**: 순백
- **헤드라인 폰트**: 굵은 산세리프 (작은 대문자 스타일, `Mining growth from a strong core business`)
- **본문 폰트**: 세리프체 (Georgia), 행간 1.5배 이상으로 여유 있음
- **강조 인용**: 굵은 이탤릭 세리프 "By far the most successful..."
- **수치**: 볼드 인라인 (`(Figure 4)`, 괄호 + 굵은 참조)

### 차트/다이어그램

- **세로 막대**: 단색(네이비) 막대. 수평 점선 그리드라인. 값 직접 레이블.
- **스캐터 플롯**: 네이비 점(•). 주요 기업명 직접 레이블. 하이라이트 영역 회색 box.
- **순환 플로우**: 화살표(삼각형 헤드) 연결 + 텍스트 노드. 원형 배치. 노드 박스 없이 텍스트 플로팅.
- **복합 다이어그램**: "Value chain adjacencies" — 박스 + 양방향 화살표 복잡 다이어그램. 중심에 "Core Businesses" 박스 강조.

## 3. BCG — Introduction (22장)

### 레이아웃 패턴

- **표지1 (포스터형)**: 세로 방향. 3D 조형물 이미지 전면 배치. 하단 타이포만 ("BUILD. CONNECT. GROW.") — 극단적 이미지 의존 레이아웃.
- **표지2 (심플)**: 흰 배경. 우중앙에 작은 타이포 "BUILD. CONNECT. GROW. / AN INTRODUCTION TO BCG" + BCG 로고 하단. 극단적 여백.
- **타이포 배경 슬라이드**: 초록 그라데이션 배경 전면 + 흰 또는 하이라이트 컬러 텍스트 블록. 벽지 타이포그래피 스타일.
- **인용 슬라이드**: 흑백 사진 우측 + 좌측 대형 인용문 (40pt+). 캡션 소문자.
- **타임라인/히스토리**: 시대별 색상 다른 카드 (원형/사각형/다이아몬드 혼합) 배치. 좌우로 연도 세로 텍스트. 좌하단 섹션 레이블.
- **섹션 브레이크**: 배경 사진/조형물 full-bleed + 대형 섹션 타이틀 백색 오버레이.
- **데이터 비주얼**: "We deliver tremendous value" — 좌측 큰 문장 + 우측 높이 다른 바 차트 (컬럼형, 다크↔라이트 그린 대비).
- **글로벌 임팩트**: 버블 레이아웃 — 지역별 원형 크기 무관 배치, 색상 계층화.
- **팀 구조도**: 좌측 역할 리스트 + 우측 교차 선 다이어그램 (관계망 시각화).
- **3단 컬럼**: PREDICTABILITY / TEAMING / OPEN COMMUNICATION — 화살표 아이콘 + 텍스트 나열.
- **목차**: 우중앙 텍스트만 + 초록 강조 키워드 인라인.
- **도넛 차트 + 대형 숫자**: "10" 대형 타이포 + 도넛 파이 차트 반반 레이아웃.
- **성장 타임라인**: 대각선 배경 그라데이션 + 수직 타임라인 (점 + 선) 오버레이.
- **로고 그리드**: 닷 그리드 배경 + 파트너 로고들 자유 배치.
- **마감 슬라이드**: 좌측 밝은 텍스트 (values 나열) + 우측 진한 초록 그라데이션 패널 분할.

### 타이포그래피/컬러

- **주색**: BCG 그린 (#00A651 계열) + 라임 그린 (#B5D334) 보조
- **다크**: 짙은 초록 (#006B3C 계열) — 히스토리 카드 배경
- **배경**: 흰색 (내부 콘텐츠 슬라이드), 초록 그라데이션 (섹션/커버)
- **헤드라인 폰트**: 경량 산세리프 (Helvetica/Avenir Light 계열) — 매우 크게, 매우 얇게
  - 예: "Our teams are open and collaborative" — 60pt 이상, regular weight
  - 예: "BUILDING IMPACT" — 대형 섹션 타이틀, thin weight, 초록 배경
- **소제목**: ALL CAPS 산세리프 (tracked out), 초록 색상 강조
- **본문**: 라이트 웨이트 산세리프 12-13pt, 행간 여유
- **인용**: 매우 큰 경량 세리프/산세리프 혼용

### 차트/다이어그램

- **히스토리 타임라인**: 색상 다른 다형 카드 (원·사각·다이아) 그리드 배치. 연도 수직 텍스트가 컬럼 구분.
- **세로 막대 (성장 스토리)**: 초록↔라이트그린 대비 막대. 수직 상승 방향 시각적 강조.
- **도넛 파이**: 대형 숫자와 결합 — "10" 타이포 + 도넛이 숫자 형태를 이룸.
- **원형 버블 맵**: 지역별 스토리를 가변 크기 원으로. 색상 그라데이션.
- **관계 선 다이어그램**: 역할-직급 리스트와 중앙 교차선 연결. 구조보다 관계의 느낌 강조.
- **아이콘 기반 3단 컬럼**: 초록 화살표 아이콘 상단 + 소제목 + 설명 텍스트.

## 5. 즉시 적용 가능한 개선 3가지

### 1. Inline Pullquote 레이아웃 추가

Bain 문서 전반에서 반복 사용되는 패턴: 본문 중간에 좌측 컬러 보더라인 + 굵은 이탤릭 인용 박스. 현재 `quote` 레이아웃은 슬라이드 전체를 차지하는 풀스크린 인용이다. `pullquote` 레이아웃(또는 `bullets`에 optional pullquote 필드)을 추가하면 텍스트 중심 슬라이드에서 핵심 메시지를 시각적으로 분리하는 컨설팅 표준 패턴을 지원할 수 있다.

구현 포인트:
- 좌측 4pt 컬러 세로 보더 (`vline` 사용)
- 우측 텍스트 영역에 굵은 이탤릭 텍스트 (14-16pt, dim 컬러)
- `bullets` 슬라이드의 `pullquote` 옵셔널 필드로 하단 배치

### 2. 경량(Light Weight) 헤드라인 스타일 옵션

BCG 아이덴티티의 핵심은 **크고 얇은** 헤드라인이다. 현재 A-Team 엔진은 모든 헤드라인에 `bold=True`가 기본이다. `font_weight: "light"` 또는 `bold: false` 옵션을 커버, section_break, big_number에 노출하면 BCG 스타일 재현이 가능하다. python-pptx에서 `r.font.bold = False`로 직접 구현 가능하다.

구현 포인트:
- spec JSON의 `headline_bold: false` 플래그 추가
- cover/section_break에 `display_weight` 파라미터 추가
- "nordic_frost"나 신규 "bcg_green" 테마에 기본값으로 설정

### 3. 텍스트+차트 복합 레이아웃 (`text_chart`)

Bain 피겨 슬라이드의 핵심 패턴: 상단 2/3는 2단 본문 텍스트, 하단 1/3은 차트. 현재 `bar_chart`는 차트만 차지하고, `two_column`은 텍스트만이다. `text_chart` 레이아웃을 추가하면 데이터-스토리 결합 슬라이드를 지원한다.

구현 포인트:
- 상단 `body_text` (단/2단 선택) + 하단 `chart` 섹션으로 슬라이드 분할
- 차트 영역 높이를 `chart_ratio` (기본 0.45)로 조절
- `figure_label` 필드로 이탤릭 "Figure N: 제목" 캡션 자동 생성
