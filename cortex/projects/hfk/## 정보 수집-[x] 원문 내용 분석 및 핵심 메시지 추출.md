주요 차이점

🎨 시각적 특성

- Modern Minimal: 구글 머티리얼 디자인 기반의 깔끔함
- Soft Pastel: 따뜻하고 친근한 파스텔톤
- Dark Elegance: 고급스러운 다크 테마
- Corporate Blue: 신뢰감 있는 전통적 비즈니스 스타일

📐 레이아웃 구조

- Modern Minimal: 균등한 16컬럼 그리드

- Dark Elegance: 조밀한 20컬럼 그리드
- Corporate Blue: 사이드바 + 메인 컨텐츠 구조

📊 차트 및 그래픽

- Modern Minimal: 데이터 중심의 심플한 차트
- Soft Pastel: 시각적으로 부드러운 도넛 차트
- Dark Elegance: 미래지향적 그라데이션 차트
- Corporate Blue: 전통적이고 명확한 막대 차트
  

■ 프롬프트

다음 문장/기사를 McKinsey, BCG, Bain과 같은 세계 최고 수준의 컨설팅 기업이 제작한 완벽한 16:9 프레젠테이션 슬라이드로 변환해주세요.

정보 설계와 비주얼 디자인 모두 최고 수준을 목표로 합니다.

  

【디자인 사양】

- 기본 색상(Primary): #2563EB (로열 블루) 및 그라데이션
- 보조 색상(Secondary): #475569 (다크 그레이)
- 액센트 색상(Accent): #f8fafc (라이트 그레이 배경)
- 중요도 표시:

  - 높음 = #ef4444 (레드)
  - 중간 = #f59e0b (오렌지)
  - 낮음 = #10b981 (그린)

- 제목: 32px, 그라데이션 효과, 볼드
- 부제목: 16px, #475569
- 섹션 제목: 18px, #1e40af, 아이콘 포함
- 본문: 14px, #334155, 줄 간격 1.4

- 헤더: 좌측 정렬 제목 + 우측 정렬 날짜/출처
- 2단 구성: 좌측 60%, 우측 40%
- 카드형 컴포넌트: 흰색 배경, 12px 둥근 모서리, 미세한 그림자 효과
- 섹션 간 적절한 여백과 계층 구조 유지

  

【콘텐츠 구성】

- 기사/문서의 핵심 내용을 초간결하게 요약
- 중요한 \*\*수치 데이터(KPI)\*\*를 시각적으로 강조 (숫자 + 단위 + 설명)

- 데이터 시각화: 바 차트, 라인 차트, 타임라인 활용
- 주요 포인트: 가장 중요한 내용만 3~4개 항목으로 요약
- 하이라이트 문장: 블루 색상으로 강조
- 복잡한 개념: \*\*"팩터 박스(Factor Box)"\*\*를 활용해 직관적으로 설명

- 시장 영향 분석
- 경쟁사 비교
- 리스크 평가 (표 형식 + 색상 구분)
- 미래 전망 / 타임라인

  

【최종 마무리】

✅ 전문적인 푸터 (로고 배치 + 출처 포함)
