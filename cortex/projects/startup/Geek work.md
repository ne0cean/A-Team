미라클레터가 N잡 연결 플랫폼 [요긱과](https://event.stibee.com/v2/click/MzMyNzEvMTAwNDY0NS8zMTMzOC8/aHR0cHM6Ly93d3cueW9naWcuY29tLw) 손을 잡고 긱워크 데이터를 분석해봤어요. 지난해 7월부터 올해 4월까지의 누적데이터를 분석한 것. 사용자 분포는 20대(25.8%), 30대(34.5%), 40대(19.3%)로 구성. 카테고리 별로 가장 많은 조회가 이뤄진 부업은 다음과 같았어요.

  

 

1위 : 온라인 설문(26.1%) - 엠브레인 / 라쿠텐인사이트

2위 : 펫시터(15%) - 와요 / 펫트너

3위 : 심부름(10.5%) - 애니맨 / 해주세요: 심부름 종류

4위 : 퀵/택배(7.2%) - 쿠팡플렉스 / 퀵커스

5위 : 데이터라벨링(6.7%) - 크라우드웍스 / 레이블러

6위 : 아이돌봄(5.3%) - 째깍악어 / 시터넷 

  

 

이렇게 많은 부업과 플랫폼이 있었군요? 요긱에 따르면 플랫폼 기준으로 가장 많았던 것은 온라인 설문 플랫폼인 엠브레인과 펫시터 서비스 와요(요긱피셜!).
