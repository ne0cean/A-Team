*서비스부문내의 SMS/MMS Capa.는 CVM팀 김준영M에게 사전 검토요청 바랍니다.(CRM은 제외)

========================================
		
		
			
[6월 전사 캠페인 공지]
			
안녕하세요, 캠페인 담당자 입니다.
			
6월 MMS/SMS 캠페인 계획을 전달드립니다.
			
1. 캠페인 기획,등록,협조, 해당 팀장님의 승인까지 이 모든 절차는 캠페인 실행일 D-1 18시 전에 반드시 완료 되야 합니다.
			
미완료시 캠페인은 미통보 삭제 됨을 알려드립니다.
			
2. 상품 출시/프로모션/이벤트/종료의 경우 사전 PLM검토가 필수 입니다.
			
스토리보드나 이벤트 스킴 등을 기획전 PLM과 협의해주세요(담당자 : 고중경팀 서민정M, 강은경M)
			
3. 대상군 추출시 주의 사항
			
- 수신동의 : 광고정보수신동의자 Only(필수) 
			
- 해당 채널 수신 거부자 필수 제외(예외없음)
			
- 중복금지 예외 및 광고정보수신 미동의자 발송 계획시 고Comm.팀과 사전 협의 필요
			
(약관 변경, 서비시 종료 공지 등 필수 안내 사항 외 예외 없음)
			
반드시 일정에 맞춰 진행 부탁 드리며, 기획하신 캠페인 제목을 바꾸지 말아 주시기 바랍니다.
			
사전 Comm. 없이 일정, 제목, 대상수(계획보다 많은) 변경이 있을시 캠페인은 자동 종료 됩니다.
			
감사합니다.
			
※ SWING 이행과 관련해 기능 개선 요청 내용 있으시면 말씀주세요.
