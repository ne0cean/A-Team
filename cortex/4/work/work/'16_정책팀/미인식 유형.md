미인식 유형
		
		
	
	
		
			
지금까지 파악된 유형을 보면 다음과 같은데 혹시 추가로 알고 있는 유형이 있으면 알려주세요
			

			
1. 바코드를 미인식하는 경우 (훼손 등이 원인으로 추정)
			
정보활용동의 페이지가 없어서 서브바코드 인식이 안되는 경우
			

			
2. 메인바코드와 서브바코드는 정상인식이 되나 기울어짐, 출력물의 품질 저하, 복사로 인한 훼손 등으로 서명 인식 실패
			

			
3. 신규계약서에 청소년 신규 동의서 스캔하는 등 신청서와 정보활용동의 페이지가 달라 서명인식 조건 불일치로 미인식 되는 경우 
			
-&gt; 접점에 변화 관리 필요
			

			
4. 스캔 이미지 기울어지거나 축소 등으로 서명인식 앵커 미인식
			

			
5. 전자신청서에 일반신청서 스캔본을 추가 처리하여 서브바코드가 있으나 실제 인식 대상이 없어서 미인식
			
-&gt; 전자신청서 추가 처리 관련 지침 안내 등 변화 관리 필요
			

			
6. 기타 특별한 사유 없이 인식 못하는 경우 (인식 솔루션 문제로 추정)
			

			

			
# 서명인식 불가 직접적 사유
			
1. 바코드 인식 불가(바코드 흐림, 바코드 낙서나 노이즈 존재, 바코드 부분 종이 접힘 등)
			
- 메인 바코드(첫 페이지) 인식 불가
			
- 서브 바코드(동의서 페이지) 인식 불가
			

			
2. 앵커 인식 불가(앵커 흐림, 앵커 부분 종이 접힘 등)
			
- 앵커는 양식지 각 모서리 상에 위치한 검은색 사각형 박스로 좌표 설정 기준값으로 서명인식이 필요한 경우 사용
			

			
3. 동의서 페이지 인식 불가
			
- 동의서 페이지 미첨부
			

			
4. 솔루션 상에서 인식 오류
			

			
# 미인식 대상 유형별 분류
			
1. 인쇄 품질 불량
			
- 프린터 토너(잉크) 부족으로 출력 상태 흐림
			
- 프린터 출력물로 노이즈가 심한 경우
			
- FAX 전송 이미지로 노이즈가 심한 경우
			
- 복사본 신청서를 사용하였는데 복사 시 이미지 확대/축소가 많이 된 경우
			

			
2. 스캔 상태 불량
			
- 스캔 시 바코드 또는 앵커 부분이 접혀서 스캔된 경우
			
- 스캔 시 각도가 일정 기준 이상으로 많이 기울어진 상태로 스캔된 경우
			
- 스캔 시 이미지 확대/축소가 많이 된 경우
			

			
3. 첫 페이지와 동의서 페이지 버전 불일치
			
- 첫 페이지는 신 버전 식지를 사용하였으나 동의서 페이지를 구 버전 식지를 사용해 동의항목이 신 버전과 다른 경우(차이 발생한 일부 동의항목 미인식)
			
- 첫 페이지는 신규 신청서이나 동의서 페이지는 명변 신청서 동의서 페이지를 사용한 경우(동의항목 전체 미인식)
			

			
4. 전자신청서 + 종이신청서
			
- 전자신청서에 종이신청서 첫 페이지만 스캔하여 추가처리한 경우(5. 동의서 페이지 없음과 동일)
			

			
5. 동의서 페이지 없음
			
- 동의서 페이지 미첨부
			
- 스캔 시 환경 설정을 양면이 아닌 단면으로 설정하여 동의서 페이지 스캔 누락(기변 신청서 해당)
			
- 스캔 시 첫 페이지 스캔되면서 동의서 페이지도 같이 밀려들어가 동의서 페이지 스캔 누락(신규, 명변 신청서 등 해당)
			

			
6. 솔루션 오류
			
- 기타 특별한 사유 없이 인식 못하는 경우
