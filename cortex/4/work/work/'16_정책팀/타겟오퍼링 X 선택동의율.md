타겟오퍼링 X 선택동의율
		
		
	
	
		
			
○ 시행목적
			
  - 내방고객 확보를 위한 각종 프로모션 고지 &quot;폼나들이 프로모션&quot;
			
  - 고객이 이벤트/ 프로모션/ 혜택 정보를 원활하게 수신하여 혜택을 놓치지 않도록
			
  - CRM 샘플확보 
			

			
○ 업무처리 방법 - 14P
			
 신청서/U.Scan 
			
   - 형광펜
			
   - 신규/기변/명변/고객요청
			
 
			
 SP 
			
    U.KEY
			

			
○ 불일치율 관리 - 신청서상의 동의/미동의 내용을 정확히 
			
  강성 VOC 발생우려
			

			
○ 업무처리 시 미인식 방지
			
※ 서명인식 불가(미인식) 대상 유형
			
1. 인쇄 품질 불량
			
- 프린터 토너(잉크) 부족으로 출력 상태 흐림
			
- 프린터 출력물로 노이즈가 심한 경우
			
- FAX 전송 이미지로 노이즈가 심한 경우
			
- 복사본 신청서를 사용하였는데 복사 시 이미지 확대/축소가 많이 된 경우
			

			
2. 스캔 상태 불량
			
- 바코드/앵커 인식 불가 : 스캔 시 바코드 또는 앵커 부분이 훼손되거나 접혀서 스캔된 경우
			
  (바코드/앵커 흐림, 바코드/앵커 상 낙서나 노이즈 존재, 바코드/앵커 부분 종이 접힘 등)
			
  *앵커는 양식지 각 모서리 상에 위치한 검은색 사각형 박스로 좌표 설정 기준값으로 서명인식이 필요한 경우 사용
			
- 스캔 시 각도가 일정 기준 이상으로 많이 기울어진 상태로 스캔된 경우
			
- 스캔 시 이미지 확대/축소가 많이 된 경우
			

			
3. 첫 페이지와 동의서 페이지 버전 불일치
			
- 첫 페이지는 신 버전 식지를 사용하였으나 동의서 페이지를 구 버전 식지를 사용해 동의항목이 신 버전과 다른 경우(차이 발생한 일부 동의항목 미인식)
			
- 첫 페이지는 신규 신청서이나 동의서 페이지는 명변 신청서 동의서 페이지를 사용한 경우(동의항목 전체 미인식)
			

			
4. 전자신청서 + 종이신청서
			
- 전자신청서에 종이신청서 첫 페이지만 스캔하여 추가처리한 경우(5. 동의서 페이지 없음과 동일)
			

			
5. 동의서 페이지 인식 불가
			
- 동의서 페이지 미첨부
			
- 스캔 시 환경 설정을 양면이 아닌 단면으로 설정하여 동의서 페이지 스캔 누락(기변 신청서 해당)
			
- 스캔 시 첫 페이지 스캔되면서 동의서 페이지도 같이 밀려들어가 동의서 페이지 스캔 누락(신규, 명변 신청서 등 해당)
