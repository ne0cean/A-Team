이전에는 파이썬이나 GO를 추천했는데 이제는 자바스크립트⁬나 Node.js 학습을 권하고 싶네요. 

하나의 컴퓨터 언어를 학습하여 이해하는 것이 프로그래밍 환경에 적응하는데 도움이 됩니다. 

그것이 C/C++, 자바, 자바스크립트, 파이썬, GO, PHP 같은 언어 중의 어느 하나여도 말이죠. 


Python과 유사한 구조의 Serpent,

C++과 유사한 Solidity,

Lisp과 유사한 LLL 등의 튜링완전언어를 통해 스마트 컨트랙트 코딩을 지원한다.


Solidity는 자바스크립트 영향을 많이 받은 언어라 자바스크립트 프로그래밍 경험이 있으면 조금 더 빨리 배울 수 있지만 필수는 아닙니다. 또한 데이터 구조 및 알고리즘, 암호 이론, 네트워크 프로그래밍 등 전산 기초 지식이 있으면 좋습니다.


수강에 필요한 역량:

• HTML 활용한 간단한 테이블 작성

• Javascript 사용한 화면의 element 제어

• node.js 사용한 npm 패키지를 다운로드 받아 호출하여 사용


블록체인 개발자면 아무래도 비트코인 소스인 C++을 학습하는 것이 좋을듯하고

dApp 개발자면 이더리움의 Solidity가 자바스크립트와 비슷하니 그쪽으로 공부하시는게 도움이 될듯하고요. 

핵심 언어가 무엇인지는 중요하지 않습니다. C/C++, 자바, Go, 파이썬, JavaScrypt 등 무엇이든지 개발할 수 있습니다.

1. 응용 프로그램 개발

- 프레임워크 학습

 

2. 모바일 개발

- Unity 엔진 프로그래밍과 C# 또는 자바스크립트 학습

 

3. 웹 개발

- Node.js와 같이 자바스크립트 또는 PHP, ASP.net, JSP(JAVA)와 같이 웹 개발에 특화된 언어 학습

4. 블록체인 개발

- 비트코인 소스인 C++ 학습과 크로스플랫폼 UI 개발을 위한 QT 프레임워크 학습

5. dApp 개발

6. 서버/네트워크 개발

7. 인공지능(머신러닝 중 딥러닝) 개발
