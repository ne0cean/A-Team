    플러그인명 주요 기능 및 특징 비고/링크
    Snapdeck

    - 자연어 프롬프트로 즉시 슬라이드 자동 생성

    - 스타일 테마 지정(다크/라이트)

    - 차트·다이어그램 자동 시각화

    - Figma 환경에서 바로 편집 및 커스터마이즈

    - Pro 요금제: 템플릿/설정 무제한

    		 5장/일 무료, Pro/Plus 요금제
    MagicSlides.app

    - 간단한 토픽만 입력하면 전체 프레젠테이션 자동 생성

    - AI 이미지 자동 첨부

    - PDF, 유튜브, URL 입력 기반 PPT 제작(곧 지원)

    - 다양한 프레젠테이션 목적(마케팅, 교육 등)에 적합

    		 커뮤니티 플러그인에서 사용 가능
    AI Slides Maker

    - 텍스트, PDF, 비디오, 웹페이지 등 다양한 입력으로 슬라이드 생성

    - 30개+ 언어 지원

    - 슬라이드별 AI 이미지 생성(Stable Diffusion, DALL·E 등 활용)

    - 프라이버시 중시, 쉬운 사용성

    		 다양한 콘텐츠 유형에 최적화
    Present.Ai

    - 프롬프트 기반 자동 슬라이드 생성

    - 스토리텔링, 비주얼, 데이터 자동 배치

    - 색상, 폰트, 레이아웃 커스터마이즈 가능

    		 차별화된 시각 효과 제공
