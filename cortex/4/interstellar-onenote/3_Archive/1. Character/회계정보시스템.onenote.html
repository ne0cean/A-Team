    (2) 특징 :

    1) Reducing handling costs & Increased processing speed

    - EDI reduces handling costs & speeds transaction processing compared to traditional paper-based processing.

    - Because EDI speeds transaction processing , the business cycle is generally shortened & year-end receivable and payable balances may be reduced


    2) Standard data format

    - EDI requires that all transactions be submitted in a standard data format

    - Translation software is required to convert transaction data from internal company data format to EDI format & vice versa


    3) Direct Links

    - EDI can be implemented using direct links between the trading partners through communication networks

    Auditing Around the Computer

    ㆍTo follow the audit trail up to the point at which accounting data entered the computer and to pick these data up again when they reappeared in processed form as computer output.

    ㆍAssumes : accurate output verifies proper processing operations.

    → pays little or no attention to the control procedures.

    ㆍIt tests normal transactions but ignores the exceptions.

    → Not generally an effective approach to auditing a computerized environment.

    Auditing Through the Computer

    ㆍTo follow the audit trail through the internal computer operations phase of automated data processing.

    ㆍTo tests the existence and functioning of control procedures.

    ㆍAssume that the CPU and other hardware are functioning properly

    → auditor’s principal tasks : verifying processing and control logic (not computer accuracy)
