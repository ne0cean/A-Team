유키 화면검색에서 "판매정책" 검색

[사용 방법]

1. 유키 하단 - 정책 등록 탭 선택

2. 새로 열리는 화면에서

제목칸 : 공지할 제목 작성

내용: 상세한 내용 작성

첨부: 하단 찾기 버튼 클릭하면, 첨부 가능

＊파일을 유키보안으로 변경 후 첨부해야 대리점 직원들이 볼 수 있음

대상자 선택 : 공지를 받을 사람 지정 가능

＊공지 받을 사람 등록하는 방법

1. 대상자 선택 버튼을 누르고 들어간 화면에서 주소록 관리 선택

2. 그룹 관리에서 그룹명에 그룹명을 입력하고 그룹 설명에 대강 적어서 저장

(예, 그룹명에 (수도권1_도매) 등)

3. 그룹 저장을 하고 상단 개인관리 탭으로 이동

4. 그룹 지정에 만든 그룹을 선택

5. 왼쪽 하단 사용자에 체크를 한 상태에서, 공지를 받을 유통망 직원이름 검색

6. 직원 이름이 조회 되면, 박스 체크하여 가운데 위치한 화살표 → 눌러서 그룹으로 이동

7. 동일한 방법으로 그룹에 들어갈 직원들을 모두 등록하고 저장

발송 채널을 SMS/메일/쪽지중 필요한 것 선택한 후 발공

- SMS 선택시 : 그룹에 등록된 직원에게 [판매정책] 제목에 입력한 내용이 문자로 발송

- 쪽지 : 선택시 등록된 팀 구성원에게 공지 내용이 발송됨
