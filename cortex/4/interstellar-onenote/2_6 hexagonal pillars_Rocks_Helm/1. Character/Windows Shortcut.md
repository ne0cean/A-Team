Windows Shortcut
		
		
	
	
		
			
Alt + 왼쪽화살표 (웹사이트 이전페이지)
			
Alt + 오른쪽화살표 (사이트 앞으로가기)
			
항목을 끌면서 Ctrl+Shift 누름(선택한 항목의 바로 가기를 만듬)
			
Ctrl+아래쪽 화살표(삽입 지점을 다음 단락의 처음으로 이동)
			
Ctrl+위쪽 화살표(삽입 지점을 이전 단락의 처음으로 이동)
			
탐색기에서 F3 키(파일 또는 폴더 찾기)
			
Alt+Enter(선택한 폴더/파일의 등록 정보 보기)
			
Alt+스페이스바(활성 창의 바로 가기 메뉴 열기)
			
Ctrl+F4(여러 문서를 동시에 열 수 있는 프로그램에서 활성 문서 닫기)
			
F6 키(창이나 바탕 화면의 화면 요소를 순환)
			
Ctrl+Shift+N 새폴더 만들기
			
Ctrl+Esc(시작 메뉴 표시)

윈도에서의 단축키 
			
Shift + F10 : 선택한 항목의 바로 가기 메뉴 표시
			
Alt + - : 프로그램의 시스템 메뉴 표시
			

			
대화상자에서의 단축키 
			
Shift + Tab : 이전 옵션으로 이동
			
Ctrl + Shift + Tab : 이전 탭으로 이동
			
Ctrl + Tab : 다음 탭으로 이동
			

			
윈도 키를 이용한 단축키 
			
시작키 + TAB : 작업 표시줄의 실행 중인 프로그램의 아이콘 선택
			
시작키 + F: 시작 메뉴의 &#39;찾기의 파일 또는 폴더&#39; 실행
			
시작키 + Ctrl + F : &#39;찾기&#39;의 컴퓨터 표시
			
시작키 + R : 시작 메뉴의 &#39;실행&#39; 메뉴 실행
			
시작키 + V : 클립보드 표시기 실행
			
시작키 + Break : [시스템 등록 정보] 대화 상자 표시 
			

			
&#39;탐색기/ 내 컴퓨터&#39; 사용 단축키 
			
Ctrl + Enter : 선택한 폴더를 내 컴퓨터를 이용해 새로운 창에서 연다.
			
F3 : 파일 찾기 실행
			
F6 : 좌우측 창간의 전환 
			

			
아웃룩 익스프레스 5.0의 주요 단축키 
			
Ctrl+M : 메일 배달
			
Ctrl+N : 새 메시지를 열거나 게시
			
Ctrl+Shift+B : 주소록 열기
			
Ctrl+R : 메시지 작성자에게 회신
			
Ctrl+F : 메시지 전달
			
Ctrl+Shift+R 또는 Ctrl+G(뉴스만) : 전체 회신
			
Ctrl+I : 받은 편지함으로 이동
			
Ctrl+&gt; 또는 Ctrl+Shift+&gt; : 목록에서 다음 메시지로 이동
			
Ctrl+&lt; 또는 Ctrl+Shift+&lt; : 목록에서 이전 메시지로 이동
			
Ctrl+U : 읽지 않은 다음 메일 메시지로 이동
			
Ctrl+Shift+U : 읽지 않은 다음 뉴스 대화로 이동
			
Ctrl+Shift+F : 메시지 찾기
