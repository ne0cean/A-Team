F2 : 포털 검색창

Ctrl + w : 탭 닫기

Ctrl + t : 새탭 열기

Ctrl + k : 현재 탭 복제

F6 / Ctrl + l / Alt + d : 주소입력창 활성화

Ctrl + 번호 : 원하는 탭으로 이동

Ctrl + Tab : 오른쪽 탭으로 이동

Ctrl + shift + n : 현재탭 새창에서 열기

Ctrl + shift + T : 종료한 탭 복구

Alt + Enter : 전체 창 전환

Alt + 방향키 : 뒤로가기 앞으로가기

Ct기 + shift + del : 인터넷을 사용하면서 남긴 정보들을 일괄 삭제
