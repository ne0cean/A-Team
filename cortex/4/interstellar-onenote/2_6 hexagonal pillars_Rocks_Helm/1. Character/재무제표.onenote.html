ROI(Return On Investment)투자수익률: NI(Net Income)당기순이익 / Assets(총자산)

= 매출액 당기순이익률 X 총자산 회전율

= (NI / 매출액) X (매출액 / Assets총자산)

ROA(Return On Assets)총자산수익률: OP(Operation Profit)영업이익 / Assets(총자산)

= 매출액 영업이익률 X 총자산 회전율

= (OP / 매출액) X (매출액 / Assets총자산)

ROE(Return On Equity)순자산수익률: NI(Net Income)당기순이익 / OE(Owner's Equity)자기자본.순자산

= 매출액 당기순이익률 X 총자산 회전률 X 재무 레버리지

= (NI / 매출액) X (매출액 / Assets) X (Assets / OE자기자본)



    ㅇ (비상장사) PER/PBR 시가순자산법과 순이익법

  • 자산가치: BPS(Book-value Per Share)주당순자산가치 X 동종업계 PBR(Price on Book-value Ratio)주당총자산비율 X 주식 수
  • 수익가치: EPS(Equity Per Share)주당순이익 X 동종업계 PER(Price Equity Ratio)주당순이익비율 X 주식 수

EPS(Equity Per Share)주당순이익 = NI / 주식 수

PER(Price Equity Ratio)주당/주가순이익비율 = 시장주가 / EPS = 시가총액 / NI

BPS(Book-value Per Share)주당순자산가치 = OE순자산.총자본 / 주식 수

PBR(Price on Book-value Ratio)주당/주가순자산비율 = 시장주가 / BPS = 시가총액 / Assets


*PBR가 1이라고 가정 할 경우, 시가총액 = OE순자산

ROE = NI / OE 는 PER = 시가총액(=OE) / NI의 역수

즉, 동종업체의 PER가 18인 경우 이의 역수인 1/18(5.6%)가 해당 업체의 ROE

회사의 ROE가 이를 상회할 경우 경영성과가 높다고 판단 가능

제 2장 원가

관리회계: 원가/가격 책정 등의 의사결정을 지원

ㅇ CVP(Cost-Volume-Profit) 분석

  • 원가-조업도-이익의 변화가 이익에 미치는 영향 분석





ㅇ ABS(Activity-Based Costing) 활동기준 원가

  • 간접비의 합리적 반영을 위해 활동을 원가의 집계 단위로 활용

ㅇ 고정비: 조업도와 매출에 무관하게 지속적으로 발생하는 비용 - 구조조정이라는 전략적 대상

ㅇ 변동비: 제품을 구성하는 재료비, 노무비, 경비와 판매비로 구성 - 원가절감의 대상

ㅇ 공헌이익/한계이익(Contribution margin): 매출-변동비(가격-개당변동비): 제품 단위당 판매가격에서 단위당 변동비 차감한 이익. 즉, 한 개를 팔 때 마다 투입되는 노력의 대가.

제품 가격을 상향하거나 제품 단위당 변동비를 절감해야 함.

ㅇ 손익분기점 BEP(Break Even Point)

  • 고정비+변동비를 매출액이 커버하여 손익이 제로에 수렴하는 지점

P X Q = F + V X Q

매출액(가격 X 수량) = 고정비 + 개당 변동비 X 수량

P X Q - V X Q = F

(P - V) X Q = F

Q = F / (P - V), 양변에 P를 곱하면

P X Q = F / (1 - V/P)

즉, BEP 매출액 = 고정비 / (1-변동비율) *(1-변동비율)=공헌이익률


가격 X 수량 = 고정비 + 적정이익 / {(한계이익)/ 가격}

P X Q = F + I {(P-V)/ P}

*한계이익=가격-개당 변동비

재무상태표(대차대조표 Balance Sheet) : (기말이라는) 특정 시점의 자산의 운용과 자산의 조달 상황. 즉, 재산의 잔고를 나타냄

손익계산서: 일정 기간의 운영으로 인한 이익(성과)의 증감

현금흐름표: 재무상태표와 손익계산의 연결을 현금의 증감으로 보여줌

  • 영업 현금흐름: NI, 발생주의 고려 몇 가지 수정 - 현금 발생하지 않은 비용, 운전 자금의 증감
  • 투자 현금흐름: 기계설비, 건물, 토지, 재무적 투자 등 취득과 매각
  • 재무 현금흐름: 은행 차입, 회사채 발행, 증자, 배당금 지급 등 자금의 조달과 상환
