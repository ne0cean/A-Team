1. 소숫점 지정 및 올림/내림/반올림
			
계산기로 계산 작업을 하다보면 화면에 많은 공간이 있는데도 불구하고&#160;
			
소숫점 몇 자리까지만 보여준다던가&#160;
			
계산기 지 멋대로 올림/내림/반올림을 한다고 생각을 하시는 경우가 있는데요.
			
그럴 때는 계산기에서 이 부분을 조정하시면 되겠습니다.
			
			
왼쪽에 있는 레버가 올림/내림/반올림을 지정하는 키이고
			
오른쪽에 있는 레버가 소숫점을 지정하는 키입니다.
			
&#160;
			
왼쪽부터 설명하겠습니다.
			
F는 영어로 Full이라고 생각하세요. (원래 F는 Full이 아닙니다.)
			
즉, 수 전체를 다 보여준다고 생각하면 됩니다.&#160;
			
제 계산기의 경우 총 12개의 수를 한 화면에 표현할 수 있습니다.
			
그러면 F를 활용하여서 8/9 (8나누기9)를 하면 어떻게 나올까요?
			
0.88888888888이라는 결과가 나오게 됩니다.&#160;
			
이는 오른쪽에 보이는 소숫점 지정 레버의 설정과 상관없이 저렇게 나오게 됩니다.
			
&#160;
			
CUT은 말 그래도 CUT입니다.&#160;
			
즉, 버림을 말합니다.&#160;
			
같은 예시로 8/9를 해보겠습니다. 그리고 소숫점은 4로 설정하겠습니다.
			
그러면 0.8888이라는 결과가 나오게 됩니다.
			
왜냐하면, 맨 뒤의 숫자가 어떻게 됬든간에 버리기 때문입니다.&#160;
			
소숫점은 4로 두었기에 소숫점 뒤에 4자리까지만 표시됩니다.
			
&#160;
			
5/4는 반올림입니다.
			
5/4로 두고 8/9와 2/9를 해 보겠습니다. 소숫점 설정은 마찬가지로 4로 두겠습니다.
			
8/9의 결과가 위에서 0.8888이라는 결과로 나왔습니다.&#160;
			
맨 마지막 숫자가 &#39;8&#39;로 끝이 났죠?&#160;
			
산수에서 0~4까지는 내림으로 계산하고 5~9까지는 올림으로 계산하죠.
			
그러니 8/9의 결과는 0.8888에서 0.8889로 변경됩니다.
			
이제 2/9를 계산해보겠습니다.
			
F로 두고 2/9로 설정하고 계산하면 0.22222222222가 나옵니다.
			
이를 5/4로 두고 계산하면 0.2222가 나옵니다.
			
&#160;
			
이제 오른쪽에 있는 소숫점 설정에 대해서 알려드리겠습니다.
			
쌀집계산기에는 소숫점 설정에 4,3,2,1,0,ADD2가 있습니다.
			
4,3,2,1,0은 4로 설정하면 소숫점 아래 숫자 4개까지 0은 정수로 표현입니다.
			
간단하죠? 하지만, F로 설정이 되어 있을 경우 이 설정은 적용되지 않습니다.
			
&#160;
			
ADD2에 대해서 알려드리겠습니다.
			
ADD2는 쉽게 말해 뭘 계산하던지 소숫점 아리 2자리를 표시하는 것을 말합니다.
			
예를 들어서 제가 400+273을 ADD2로 설정하고 계산을 하면
			
673이라고 나오지 않고 6.73이라고 나오게 됩니다.&#160;
			
81-21을 계산하게 되면 60이라고 나오지 않고 0.60이라고 나옵니다.
			
&#160;
			
2. 계산기로 마진 계산? COST/SELL/MAR
			
계산기를 통해서 상품의 마진당 구입금액을 설정할 수 있습니다.
			
자, 여러분이 지금부터는 병용전자(안산 각골초점)의 주인입니다. (가상의 공간입니다.)
			
가게에 로지텍 G PRO 키보드가 입고되었습니다.
			
&#160;
			
현재 G PRO키보드의 네이버 최저가는 97,000원(2020/02/19 기준)입니다.
			
병용전자(안산 각골초점)에서는 네이버 최저가로 상품을 받는다고 가정합시다.
			
점주의 입장에서 여기에 마진 10%를 붙이고 VAT는 별도로 지정하겠습니다.
			
이제 판매가를 구해봅시다.
			
&#160;
			
이것을 계산기로 쉽게 계산을 하기 위해서는 아래 키를 이용합시다.
			
			
&#160;
			
COST는 원가를 말합니다.
			
제가 네이버 최저가로 상품을 받으므로&#160; 97,000을 누르고 COST를 누릅니다.
			
MAR은 마진을 말합니다.&#160;
			
마진을 10%으로 지정해야 하므로 10을 누르고 MAR을 누릅니다.
			
&#160;
			
그러면, 계산기를 기준으로 원가와 마진 설정이 되었습니다.
			
이제 판매가를 구합시다.&#160;
			
SELL은 판매가를 말합니다. 그냥 SELL을 누르면 됩니다.
			
107,777이라는 결과가 나옵니다.&#160;
			
&#160;
			
3. 부가세를 계산기로 계산한다? TAX+와 TAX-, RATE SET
			
아직 여러분은 병용전자(안산 각골초점)의 점주입니다.
			
그리고 이번에는 G PRO키보드 판매금액에 부가세를 포함해보겠습니다.
			
&#160;
			
그전에 계산기에 부가세의 퍼센트를 지정해야 합니다.
			
대한민국 국세청에 부가세의 퍼센트는 판매금액에 10%입니다.
			
계산기에 부가세의 퍼센트를 지정하기 위해서는 RATE SET을 길게 꾸욱 누르고 있어야 합니다.
			
그러면 아래와 같이 화면이 변합니다.
			
			
저의 경우 이미 10%로 지정을 하여서 저렇게 나옵니다.
			
여러분의 경우 다르게 설정이 되어 있을 경우 10을 누르고 AC버튼을 누르면 됩니다.
			
&#160;
			
이제 G PRO키보드의 부가세를 포함하는 가격을 구해보도록 하겠습니다.
			
107,777이라는 VAT별도의 금액을 계산기에 입력합니다.
			
그리고 TAX+를 누릅니다. 118,554라는 결과가 나오는데요.&#160;
			
이것이 G PRO키보드의 부가세 포함 가격이 됩니다.
			
여기서 TAX+를 한 번 더 누르게 되면&#160;
			
10,777이라는 결과가 나오게 되는데요.&#160;
			
이게 국세청이 가져가는 부가세입니다.&#160;
			
&#160;
			
TAX-는 말 그대로 세금을 뺀 가격을 보여줍니다.
			
118,554에서 TAX-를 입력하면 107,777이라는 결과가 나옵니다.
			
그리고 여기서 한 번 더 눌러도 순수 부가세만 나타나게 됩니다.
			
&#160;
			
마무리를 짓겠습니다.&#160;
			
병용전자(안산 각골초점)에서 G PRO키보드의 가격은&#160;
			
108,000원으로 지정하겠습니다.
			
이런식으로 여러분이 매장을 운영을 할 때도 이 전자 계산기만 있어도 어느 정도 왠만한 계산은 쉽게 마무리 됩니다.
			
&#160;
			
3. 컴퓨터 견적시 도움이 되는 계산기 기능? GT기능
			
계산기에서 GT버튼과 계산을 완료하고 &#39;=&#39;을 누르고 나면 GT라고 나오는 것을 볼 수 있습니다.
			
			

			
			
&#160;
			
게산기에서 GT는 Grand Total의 약자입니다.
			
총 합계라는 뜻입니다.&#160;
			
제가 예를 들어서 10을 누르고 &#39;=&#39;을 누르고 20을 누르고 &#39;=&#39;을 눌렀다고 가정합시다.
			
이게 계산기에 GT버튼을 누르면 30이라는 값이 나오게 됩니다.&#160;
			
이는 계산 값의 결과인 10과 20의 합계입니다.
			
즉, 계산값의 총 합계를 보여주는 것입니다.
			
&#160;
			
컴퓨터 견적을 짤 때&#160; 이 기능이 유용한 이유는 배송비 때문입니다.
			
컴퓨터 부품은 적어도 5개의 부품이 있어야 작동을 하게 됩니다.&#160;
			
최대 부품의 개수는 정해저 있지 않습니다.&#160;
			
&#160;
			
3가지의 부품을 각각 다른 곳에서 구입을 한다고 가정해봅시다
			
CPU는 제이씨현에서 150,000원에 배송비는 2,000원입니다.
			
RAM은 삼성전자에서 50,000원에 배송비는 1,500원입니다.
			
마더보드는 웨이코스에서 74,000원에 배송비는 3,000원입니다.
			
이 모든 것을 한 번에 계산하기 위해서는 아래와 같이 입력하면 됩니다.
			
&#160;
			
150,000/+/2,000/=
			
50,000/+/1,500/=
			
74,000/+/3,000/=
			
C/GT
			
이렇게 입력하면 됩니다.
			
그러면 280,500원이라는 결과가 나옵니다.&#160;
			
GT값을 지우기 위해서는 AC키를 누르면 됩니다.
			
&#160;
			
4. 퍼센트 계산
			
대부분 아실 것 같지만 저는 처음 사용할 줄 몰랐기에 알려드리겠습니다.
			
&#160;
			
1,000의 10%는 100입니다. 알죠? 이 결과를 계산기로 나오게 해보겠습니다.
			
이 때는 계산기의 &#39;%&#39;키를 사용하면 됩니다.
			
			
&#160;
			
위와 같은 결과를 얻기 위해서는 아래와 같이 키를 입력하면 됩니다.
			
1,000/X/10/RATE SET %
			
이렇게 입력하면 결과로 100이 나오게 됩니다.
			
&#160;
			
5. 계산기에 메모리가 있다? MRC/M+/M-
			
			
계산기에는 치명적인 단점이 있는데요.
			
계산식에 덧셈과 곱셈이 있을 경우 무엇부터 계산을 해야하는질 모릅니다.
			
그냥 식의 왼쪽부터 오른쪽으로 순차적으로 계산을 하게 됩니다.
			
그럴 때 이 기능을 사용하면 됩니다.
			
&#160;
			
1. M+
			
M+는 어떤 값을 양수로 메모리에 저장하는 기능을 말합니다.
			
예를 들어서 4X3+2X6=24입니다.
			
하지만 계산기로 계산하면 84가 나옵니다.
			
&#160;
			
이럴 때 24가 나오게 계산을 하려면 아래와 같이 입력하면 됩니다.
			
4/X/3/M+
			
2/X/6/M+
			
MRC
			
로 입력하면 24가 나오게 됩니다.&#160;
			
그냥 쉽게 말해서 식에 괄호를 친다고 생각되겠습니다.
			
&#160;
			
2. M-
			
M-는 어떤 값을 음수로 메모리에 저장하는 기능을 말합니다.
			
예를 들어서 4X3-2X6=0입니다.
			
하지만 계산기로 계산하면 60이 나옵니다.
			
&#160;
			
이럴 때 0이 나오게 계산을 하려면 아래와 같이 입력하면 됩니다.
			
4/X/3/M+
			
2/X/6/M-
			
MRC
			
로 입력하시면 0이 나오게 됩니다.
			
&#160;왜? 4X3식에는 M+를 붙이고 2X6에는 M-를 붙였냐면&#160;
			
&#160;
			
만약에 제가 양쪽에 다 M-를 붙였다면 이렇게 계산이 됩니다.
			
-(4X3)-(2X6)=-24가 나옵니다.
			
&#160;
			
3. MRC
			
사실, MRC키는 메모리 기능이 내장된 일부 계산기에만 탑제가 되게 됩니다.
			
제 쌀집 계산기의 경우 이 키가 존재하는데요.
			
일부 다른 계산기에서는 MR/MC로 따로 키가 있는 경우가 있어요.&#160;
			
&#160;
			
그래서 MRC에 대해서 알려드리면
			
불러오기와 제거라는 뜻입니다.&#160;
			
MRC를 한 번 누르면 메모리에 저장되어 있던 값이 불러와지고&#160;
			
두 번 누르면 메모리에 저장되어 있던 값이 제거 됩니다.
			
&#160;
			
6. AC와 C의 차이점?
			
			
계산기에 AC와 C의 차이점을 잘 모르시는 분들이 있더라구요.
			
그래서 추가로 알려드리겠습니다.
			
&#160;
			
AC는 All Clear의 약자로&#160;
			
쌀집 계산기의 경우 이 키를 전원ON키와 같이 사용됩니다.
			
그래서, 어떻게 보면 가장 자주 쓰이는 키이기도 합니다.
			
AC는 메모리 뿐만 아니라 모든 계산기의 데이터를 공정초기화(공장에서 출고되는 상태로 만듬)를 시킵니다.
			
즉, 메모리의 저장데이터와 GT값, COST, SELL, MAR 등이 지워집니다.
			
하지만, 위에서 언급한 지정한 세율의 값은 변하거나 지워지지 않습니다.
			
&#160;
			
C는 Clear의 약자입니다.
			
그냥 계산기 화면에 표시되어 있는 숫자의 값만 지워지게 됩니다.
			
메모리 및 일부 저장 데이터는 지워지지 않습니다.
			
&#160;
			
(쿠키) 계산기 자동꺼짐 기능?
			
쌀집 계산기의 치명적인 단점인데요.
			
자동꺼짐 기능이 내장되어 있습니다.
			
&#160;
			
계산기의 방전을 막기 위해서 해 놓은 기능으로 판단됩니다.
			
아무리 태양광으로 충전을 한다고 해도 충전 전류의 한계가 있기 때문에&#160;
			
어두운 공간에서 장시간 사용시에 방전의 위험이 있습니다.
			
&#160;
			
이 자동꺼짐 기능은 계산기를 조작하지 않고 일정시간 계산기를 방치할 경우&#160;
			
&#39;CASIO&#39;라는 로고와 함께 계산기가 자동으로 꺼지게 됩니다.

2*3+3*6+4*4 를 계산할 때 2*3=, 3*6=,4*4= 을 친후 GT를 누르면 총합이 계산됩니다. 이 기능은 현재가치 계산시에도 유용하게 쓰입니다.

2*3M+,3*6M-, 4*4M+, MR ---&gt; 이런면 총합이 계산이 됩니다.
