Joyn.T 1.0 개선 사항

  1. 로딩화면(대기시간) 개선
  2. Joyn.T 구동 시 로딩화면(Joyn.T 로고)을 자신이 설정한 사진이나 별도의 정보 제공성 화면으로 대체하여 서비스 이용시 User들의 체감 대기시간을 줄이는 것이 필요함.
  3. 폰 주소록 그룹 - Joyn.T 그룹 연동
  4. 폰 기본 탑재 주소록의 그룹을 조인과 연동하여 조인에서 별도의 그룹을 생성해야하는 불편을 개선해야 함.
  5. 그룹내 메시지 발송대상 설정
  6. 현재 Joyn.T에서 생성한 그룹은 가변적이지 않아서, 발송대상을 가감해야 할 경우

    7. 다시 설정해야 함.

  7. 영상통화 전면 카메라 활성화
  8. 현재 Joyn.T를 활용한 영상공유 기능은 브로드캐스팅에는 손색없는 화질이나

    9. 후면카메라만을 사용하여, 상대방의 얼굴을 보며 통화하는데 어려움이 있음.



Joyn.T 활용한 통화 시 추가 기능 - Joyn.T를 활용한 통화에만 사용할 수 있도록 하는 부가서비스

  1. 통화 중 BGM 기능
  2. 연인과의 통화 등 가까운 사람과의 통화에 BGM을 설정할 수 있도록하는 기능.
  3. "말걸지마" 모드
  4. 메시지는 허용하되 전화만 일시적으로 수신하지 않는 상태로 전환
  5. 현재 기분상태를 설정 가능하도록하여 발신자에게 아이콘이나 이미지 형태로 보여주는 기능
  6. Photo-ring 서비스
  7. 영상통화 서비스가 활성화 되지 않는 큰 원인 중 하나는, 자신의 현재 상태(외모 등)을

    8. 상대방에게 보여주길 쑥쓰러워하거나 꺼려하는데 있음.

  8. 이를 영상형태가 아닌, 자신이 통제력을 가질 수 있고 보다 익숙한 형태인 이미지로 전환하여

    9. 상대방에게 전화를 걸었을 때, 자신이 보여주고 싶은 사진이 뜨도록 하는 기능

  9. 화면 공유 기능
  10. PC의 원격제어 처럼, 통화중 상대방과 같은 화면을 볼 수 있도록 하는 기능
  11. 음성변조 콜
  12. Fun요소 가미를 위해 발신 시나, 통화 중간에 다양한 목소리로 음성을 바꿔서 통화할 수 있도록 하는 기능
  13. 대화 참여 기능
  14. 통화중에 전화가 오면 해당 발신자를 대화에 참여시켜서 그룹통화로 전환하거나,

    15. 통화중에 제 3자를 불러들여서 함께 통화하는 기능

  15. 다자간 Comm. 상황에서 정보를 양자간에 확인하거나 전달하기 위해 전화를 끊고 다시 걸고,

    16. 다시 제 3자에게 전화해야 하는 불편함을 해소할 수 있음.

    3. 유저들이 직접 스티커를 제작하고 이를 사용하며 스티커 장터에서

  4. 기 설정된 그룹이나 채팅방에서 몇 명에게만 선택적으로 메시지를 발송 할 수 있도록하는 기능

    5. (현재 Joyn.T에서는 한번 설정한 그룹이 픽스되어, 전송 대상을 가감 하려면 다시 채팅방을 생성해야 함)

  6. T끼리 요금제에 가입한 고객들이 1회 인증절차만 거치면,

추가 제언

  1. 고객들의 Comm. 서비스 이용 Habit 전환 - "바로 옆 사람과 대화하는 것처럼"
  2. T끼리면 번호를 몰라도 통화나 메시지 전송이 가능하도록 하는 기능구현.
  3. 일반적인 대화상황에서는 근거리에 있으면, 상대방을 처음보거나 이름을 몰라도 대화가 가능함.

    4. 그러나, 통신서비스에서는 Key 값으로 010으로 시작하는 식별번호를 알아야만 상대방과 Comm.이 가능함.

  4. 근거리에 있는 당사 고객간 : 위치정보 + 프로필 정보(소셜폰북과 연계) 검색으로 통화나 메시지를 전송할 수 있도록 하는 기능
  5. 원거리에 있는 당사 고객간 : 이름 + 프로필 정보 검색 후 Comm. 할 수 있도록 하는 기능.

    6. -> 전화번호를 몰라도, T끼리면 언제 어디서나 누구나 서로 Comm. 가능 (막대한 가입자 기반의 강점 활용)

    -> 근처에 있는 다른 사람의 프로필 정보(직장, 출신, 취미&선호 등)에 대한 호기심과,

    자신의 프로필을 과시하고 싶어하는 욕구를 동시에 충족시키는 것이 가능함.


  1. PC 전화
  2. PC에 추가적으로 동글이나 소형 단말을 연결하면 이 기기가 핸드폰과 - PC를 연동시켜 데이터망을 사용한 통화가 가능 하도록하는 기능
  3. PC에 Joyn.T 메신저를 설치하게되면, 채팅 중 통화가 필요한 경우 다이렉트로 전화까지 할 수 있도록 하기 위함.

  4. 마케팅 방향
