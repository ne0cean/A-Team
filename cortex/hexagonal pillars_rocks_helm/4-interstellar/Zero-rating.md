▶ 동영상이 부담이 가장 크긴하지….

제로레이팅 서비스에 대한 소비자들의 만족도는 대체로 높았고, 통신비 인하 효과에 대해서 긍정적인 반응을 나타냈다. 제로레이팅은 서비스 가입자 신규유치는 물론 기존 이용자의 사용량 확대도 이끌어내는 효과가 있었다.

현재 이용하는 서비스가 제로레이팅을 제공할 경우에 서비스 이용시간을 늘릴 것이냐는 설문에, 응답자들은&#160;최소 5.1%포인트(P)에서 최대 14%P까지 이용시간을 늘리겠다고 답했다.

제로레이팅 제공 가정 시 이용 비중 변화 정도

![image](https://graph.microsoft.com/v1.0/users('realpage@naver.com')/onenote/resources/0-d62e03e210bd7283956182309dcff5ff!1-733661839CC53BA5!7931/$value)

기존에 해당 서비스를 이용하지 않는다고 답한 응답자 중 50~60% 가량도 &quot;제로레이팅 제공 시 새롭게 해당 서비스를 이용할 것&quot;이라고 응답했다.&#160;신규 유입 및 이용 증가 측면에서 모두 유의미한 변화를 나타냈다.

제로레이팅에 대한 소비자 만족도는 대체적으로 높은 것으로 조사됐다. 서비스 종류에 관계없이 이용만족도는 50%가 넘었다. 멜론(음원), 옥수수(동영상), 지니, 올레TV모바일의 경우 제로레이팅 서비스에 대한 만족도가 각각 53.7%, 58.7%, 63.5%, 64.2%였다. 불만족도는 7% 미만으로 매우 낮게 나타났다.

◦(경쟁이슈) 상대적으로 통신사에 비해 콘텐츠/플랫폼 사업자 측면에서 전용 데이터 제공이 이용증가에 높은 영향을 주는 것으로 나타남 －통신사 측면에서 전용 데이터 서비스가 가입자 유지 및 신규 가입자 유치에 미치는 영향은 높게 나타나고 있지 않으나, 단위 시간 당 데이터 소비량이 높은 동영상 서비스의 경우 에는 상대적으로 높은 영향을 주는 것으로 나타남

데이터 비용 절감 측면에서도 서비스 종류에 관계없이 전반적으로 서비스 가입자들의 50% 이상이 &quot;제로레이팅을 통해 데이터 비용을 절감했다&quot;고 인식하고 있었다.&#160;다만 데이터 비용 절감 인지여부에 대해서 잘 모르겠다고 응답한 비율도 40∼50% 정도로 나타났다.

□ 전용 데이터 제공 서비스 이용 만족도 ◦전용 데이터 제공 서비스 가입자들의 경우 전반적으로 서비스 종류에 관계없이 이용 만족도가 50% 이상으로 높은 편으로 나타남 －멜론, 옥수수, 지니, 올레TV모바일의 경우 관련 전용 데이터 제공 서비스에 대한 만족 도가 각각 53.7%, 58.7%, 63.5%, 64.2%인 반면, 불만족도는 7% 미만으로 매우 낮게 나타남 ◦데이터 비용 절감 측면에서도 서비스 종류에 관계없이 전반적으로 전용 데이터 제공 서비스 가입자들의 50% 이상이 해당 서비스 이용을 통해 데이터 비용을 절감한 것으로 인식하고 있음 －멜론, 옥수수, 지니, 올레TV모바일의 경우 설문결과 각각 41.7%, 43.6%, 55.4%, 50.4%의 이용자들이 데이터 비용을 절감한 편이라고 응답 －반면, 각 서비스 별로 데이터 비용 절감 인지여부에 대해서 잘 모르겠다고 응답한 비율도 40∼50% 정도로 나타나고 있음 ◦현재 요금제의 데이터 제공량에 대한 만족도 설문에서도 전반적으로 전용 데이터 서비스를 이용 중인 가입자들이 상대적으로 높은 만족도를 보임 －정액요금제 가입자의 음성, 데이터 제공량의 만족도는 5점 척도 기준으로 각각 평균 3.90과 3.27로 나타나 상대적으로 음성 제공량의 만족도가 높게 나타남 －전체 평균과 전용 데이터 제공 서비스 가입자의 음성, 데이터 제공량 만족도를 비교해 보면, 전용 데이터 제공 서비스 가입자의 음성 제공량 만족도는 3.97로 큰 차이를 보이지 않는 반면, 데이터 제공량의 만족도는 3.49로 약 0.22 가량 높은 것으로 나타남 －이러한 차이는 전용 데이터 제공 서비스 이용이 특정 서비스 사용에 필요한 데이터 부담을 경감하면서 요금제에서 제공하는 데이터 제공량의 가용 폭을 넓힌 결과라고 볼 수 있음
