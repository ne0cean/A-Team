### 왜 K-Pop 콘서트가 최강의 Day 1 Retention Use Case인가?



1. 발생 빈도 + 규모

→ 2025년 서울에서만 BTS·블랙핑크·뉴진스·세븐틴·에스파 등 대형 콘서트 120회 이상

→ 한 번에 1~6만 명 모임



2. 감정 강도 극대화

→ “같이 떼창하고 싶은 사람”, “포카 교환하고 싶은 사람”, “같이 앵콜 뛰고 싶은 사람”이 수만 명인데 끝나고 나면 영원히 못 만난다



3. 자연스러운 바이럴

→ 콘서트 끝나고 트위터에 “오늘 ○○ 콘서트에서 만난 언니/오빠 덕분에 인생 공연됨 ㅠㅠ 앱으로 연결됐어요” → 순식간에 10만 다운로드



4. 유료화 포인트 명확

→ “같은 편(좌석 구역) 사람만 보기”

→ “포카 교환 원하는 사람 필터”

→ “콘서트 끝나고 2시간 동안만 채팅 가능” (이걸 프리미엄으로)



### 그래서 진짜 MVP v3 (3주 안에 가능, 진짜로 팔리는 버전)



앱 이름 후보: “같이 떼창” 또는 “콘네트” (Con + Connect + Concert)



1. 앱 키면 → “다가오는 공연” 리스트 (티켓몬스터·인터파크 API 연동)

2. 내가 가진 티켓 선택 → 자동으로 해당 공연 채널 입장

3. 공연 당일 현장 들어가면 → 반경 1km 내 같은 공연 티켓 가진 사람만 리스트업

4. 프로필에 자동으로

→ 좌석 구역 (예: “A구역 15열”)

→ 최애 멤버 배지

→ “포카 교환 원해요 / 응원봉 2개 있어요 / 같이 앵콜 뛸 사람” 한 줄

5. 탭하면 바로 채팅 (템플릿 5개 제공)

6. 공연 끝난 후 2시간 동안만 채팅 가능 → 그 이후 자동 소멸 (FOMO 극대화)



이거의 100% 확률로

첫 타깃 공연 하나만 제대로 잡아도 (예: 2026년 3월 에스파 서울 콘서트 6만 명)

→ 당일 2~3만 다운로드 + Day 1 Retention 85% 이상 나온다.
