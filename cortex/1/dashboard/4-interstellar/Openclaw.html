ERRC (Eliminate–Reduce–Raise–Create)

👉 기준 = 시장 가치곡선 재설계

- 무엇을 없앨까 (Eliminate)

- 무엇을 줄일까 (Reduce)

- 무엇을 높일까 (Raise)

- 무엇을 새로 만들까 (Create)




🟣 STAR (Start–Try–Amplify–Recover)

👉 기준 = 행동 실험 사이클

- Start → 시작할 것

- Try → 실험할 것

- Amplify → 잘 되는 것 확대

- Recover → 실패 교정
