모바일 상품권
		
		
	
	
		
			
1. 유키등록
			
- 타 사은품과 동일하게 개통 D+1에 발주 확인됨 / 발주 확인 당일 문자* 발송 진행
			
(*모바일상품권별 문자 내용 첨부하였습니다.)
			

			
2. 교환 Process
			
1) 신세계, 홈플러스모바일상품권
			
- 수신받은 MMS로 상품권샵에서 지류상품권으로 교환처리 후 사용 가능
			
2) 롯데모바일상품권
			
- 수신받은 LMS 내의 ARS 번호로 연결하여 인증메시지(MMS) 수신
			
- 해당 MMS로 상품권샵에서 지류상품권으로 교환처리 후 사용 가능
			

			
3. 모바일상품권 유의사항
			
- &#39;17년 1월 현재, MMS로 지류상품권 교환 시 수신자의 주민등록증 확인 하는 것이 원칙
			
- 모바일상품권 교환 유효기간은 1년으로 설정되어 있으며,
			
발송 후 1년되는 시점까지 미사용 시 시스템상 자동 반품처리 되고 있음
			
(미사용 고객이 유효기간 경과 후 재발송 요청할 경우 유키 상 재지급 처리하여 발송해주고 있음)
			
- 지류상품권으로 교환 후 반품처리는 불가함.
			
(최초 발송은 모바일로 진행되었으나, 반품시 지류상품권으로 입고되어 재고처리 어려움이 있음)
			
- 재발송 요청 시, SMS로 재발송 되고 있으며 유효기간은 최초 발송 유효기간과 동일함
			
예) &#39;16년 1/1 최초 발송 후 3/1 재발송 요청 시, 동일하게 &#39;16년 12/31까지의 유효기간으로 발송됨
			

			
4. 정산 Process
			
- 정산기준 : 매월 1일~말일 교환 및 반품완료 기준
			
* 모바일상품권은 교환 시 배송완료 처리됨. (교환 전까지는 처리값 없는 상태로 정산 미포함)
