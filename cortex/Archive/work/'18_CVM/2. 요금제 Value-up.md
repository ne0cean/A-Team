2. 요금제 Value-up
		
		
	
	
		
			

		
		
			
ㅇ Mission : 고객 속성 기반 요금 Value-up 
			
  ① 유통망 : 일반업무 고객 대상(월 평균 130만) Value-up
			
  ② 고객센터 : [Target: Up60%(+17.4%)]
			

			
(부문장님 feedback) 기변 고객 분석을 통해 target별 value package 마련하고 offer 진행되어야 함.
			
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160; &#160;&#160;&#160;   유통망에서 일반업무 고객 대상 offer는 직원 인센티브 구조상 실행 어려움.  
			
            회사 차원의 인센티브와 개인 차원의 인센티브의 align되지 않아 유선 bundle-up, NUGU 판매는 가능하지만 요금제 up-sell은 불가
			
&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;&#160;   &#160; &#160;&#160;&#160;모든 캠페인을 모든 채널에서 실행할 수는 없기 때문에 캠페인의 기획/의도와 접점 채널간의 매칭을 고려할 것
