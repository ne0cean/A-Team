■ 동시판매 개념 확장

1) 가족단위 동시판매 (확장형 동시판매)

유선 명의자 본인은 유선만 가입하고 기변 니즈가 없으나, 가족 구성원 중 타사 이동전화를 사용하는 고객을 번호이동 시켜서 유무선 동시 판매(결합)

유선 : A(신규가입), 무선 : B(MNP)

→ 레슨이 있을 경우, 유통망 동시판매에도 적용이 가능한 부분

2) 기존 결합 가입자 : 가족단위 유무선 Cross Value-up

기존 유무선결합가입자 중 유선명의자가 아닌 가족 구성원 중 (자녀나 부모님) 기변 니즈가 있는 경우

1. 유선 : Up-sell (IPTV 가입 or 다셋탑) + 무선 : 다른 가족구성원 기변

2. 유선 : Value-up(Giga전환 or UHD) + 무선 : 다른 가족구성원 기변
