단축키 
			
Ctrl(cmd)+Shift+/를&#160;누르시면&#160;여러&#160;단축키들을&#160;쉽게&#160;둘러보실&#160;수&#160;있습니다.
			
프로토타입 탭 (Shift+E)
			
디자인 화면과 프로토타입 화면을 빠르게 이동하는 단축키&#160;shift + E
			

			
원은 단축키 O
			
라인은 단축키 L 버튼
			
사각형은 단축키 R
			

			
피그마 폰트 조절 단축키, 익숙해지면 아주 유용할 것 같다. 크기, 굵기, 행간, 자간.. 순으로 손으로 외워두자. (cmd = ctrl, option = alt)
			
Font size : 크기 -&#160;cmd + shift + &lt; &gt;
			
Font weight : 굵기 -&#160;cmd + option + &lt; &gt;
			
Line height : 행간 -&#160;option + shift + &lt; &gt;
			
Letter spacing : 자간 -&#160;option + &lt; &gt;
			

			

			
&#160;프레임의 크기를 줄일 때 Constraints의 영향을 받지 않으려면 Cmd(Ctrl)을 함께 누르고 사이즈를 줄이면 된다.
