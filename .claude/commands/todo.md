메모 파일 경로: `$HOME/.claude/todo-memos.md`

## 서브커맨드 처리

**인수 없이 `/todo` 만 입력한 경우** — 현재 메모 목록 출력:
파일을 읽어 번호·내용·날짜를 표 형식으로 출력한다.
메모가 없으면 "메모 없음 — `/todo [내용]` 으로 추가하세요" 출력.

**`/todo [내용]` — 메모 추가:**
파일에 아래 형식으로 한 줄 append한다:
```
- [ ] [내용] · {오늘날짜 YYYY-MM-DD}
```
추가 후 "✅ 메모 추가됨: [내용]" 확인 출력.

**`/todo done [번호]` — 완료 처리:**
해당 번호의 `- [ ]` 를 `- [x]` 로 변경하고 뒤에 ` ✓` 를 붙인다.
"✅ [번호]번 메모 완료 처리됨" 확인 출력.

**`/todo del [번호]` — 삭제:**
해당 번호 줄을 파일에서 완전히 제거한다.
"🗑 [번호]번 메모 삭제됨" 확인 출력.

**`/todo clear` — 완료된 항목 일괄 삭제:**
`- [x]` 로 표시된 줄을 모두 제거한다.
"🧹 완료된 메모 N건 삭제됨" 확인 출력.

## 메모 파일 형식

```markdown
# Todo Memos

- [ ] connectome TURN 서버 설정 전에 coturn 비용 검토 · 2026-03-20
- [ ] longform API 키 할당량 다 차면 xAI로 폴백 로직 확인 · 2026-03-20
- [x] vibe-toolkit GitHub Push · 2026-03-19 ✓
```
